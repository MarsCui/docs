# 快速部署

## 前言

| 项目   | 内容                                       |
| ---- | ---------------------------------------- |
| 适用角色 | 普通用户                                     |
| 导航路径 | 模型服务 > 快速部署                              |
| 功能定位 | 用户从云上模型库筛选目标模型，通过智能推荐算力完成部署任务的快速开始入口 |

## 页面结构

"快速部署"由 3 步流程组成：① 模型库筛选 → ② 智能推荐部署 → ③ 发布部署弹窗。

## 操作步骤

### Step 1：选择云账号与模型

1. 进入平台首页，点击左侧导航栏的 **"模型服务 > 快速部署"** 菜单，进入"模型库"页面。
2. 页面顶部标题"模型库"与提示语"选择可部署云账号与地域后，筛选并部署匹配模型"，右上角提供 **"刷新资源"** 按钮。
3. 左侧"可部署范围"卡片默认显示 **"当前范围 阿里云 / 华东 2（上海）"** 标签，并提供 **"范围筛选"** 下拉（全云 / 全地域）。
4. 左侧云厂商列表展示可部署范围（每个卡片含云厂商 logo + 名称 + 地域）：
   - **华为云**·华东-上海一
   - **亚马逊**·欧洲（法兰克福）
   - **阿里云**·中国香港
   - **阿里云**·华东 2（上海）（默认选中，紫色高亮）
   - **阿里云**·新加坡
   - **谷歌云**·东京
   - **谷歌云**·爱荷华
5. 点击目标云厂商卡片（如"阿里云·华东 2（上海）"），系统将基于该范围筛选匹配的模型。
6. 右侧"匹配模型"区域支持按 **"全模型"** / **"全系列"** / **"全场景"** 下拉筛选 + **"搜索"** 输入框 + **"默认排序"** 按钮筛选模型。默认显示 9 / 32 个模型，示例模型：
   - **Qwen3-8b**（对话模型 / 文本 / 工具调用 / 深度思考 / 文本生成，**阿里云 / 华东 2（上海）**）：**"详情"** / **"部署模型"**
   - **DeepSeek-R1**（对话模型 / 文本 / 工具调用 / 深度思考 / 文本生成，**阿里云 / 华东 2（上海）**）：**"详情"** / **"部署模型"**
   - **Qwen3-8b**（对话模型 / 文本 / 工具调用 / 深度思考 / 文本生成，**阿里云 / 华东 2（上海）**）：**"详情"** / **"部署模型"**
   - **Qwen3-8b**（对话模型 / 文本 / 工具调用 / 深度思考 / 文本生成，**阿里云 / 华东 2（上海）**）：**"详情"** / **"部署模型"**
   - **DeepSeek-V3.2**（对话模型 / 文本 / 工具调用 / 深度思考 / 文本生成，**阿里云 / 华东 2（上海）**）：**"详情"** / **"部署模型"**
   - **-**（多模态无标题模型，**阿里云 / 华东 2（上海）**）：**"详情"** / **"部署模型"**
   - **Qwen3.5-27b**（对话模型 / 文本 / 图片 / 视频 / 工具调用 / 文本生成，**阿里云 / 华东 2（上海）**）：**"详情"** / **"部署模型"**
7. 点击目标模型的 **"部署模型"** 按钮，进入 Step 2 智能推荐部署。

### Step 2：智能推荐部署

1. 点击模型行的 **"部署模型"** 按钮后，进入"智能推荐部署"页（页面左上角显示返回箭头 + **"智能推荐部署"** 标题）。
2. 页面顶部展示 **"已选模型"** 信息卡（图标 + 模型名 **Qwen3-8b** + 系列 **Qwen3** + 能力 **文本生成** + 版本 **1.0.0** + 上下文 **128**），右上角提供 **"部署模式"** 单选 Tab（**单节点** / **高可用**），其中"单节点"已选中并展示 **"验证、低并发、成本"** 提示。
3. **"推理引擎"** 配置区（说明：用哪种推理框架来部署模型），单选目标推理引擎：
   - **vLLM**（吞吐优先、生态成熟、OpenAI API 兼容，默认选中，紫色高亮）
   - **SGLang**（复杂推理链路、多轮编排与缓存命中优先）
4. **"业务策略"** 配置区（说明：按价格、性能、库存与显卡数量调整候选排序），单选目标业务策略：
   - **经济适用**（按小时单价与资源利用率排序，默认选中，紫色高亮）
   - **卓越性能**（优先高性能 GPU、高带宽与最新引擎）
   - **现货速达**（优先可立即启动的空闲资源）
   - **显卡数量**（优先满足多卡并行与扩展需求）
5. **"模型算力匹配方案"** 卡片（右上角"推荐"标签 + **Top 1** 徽标 + **"确认部署"** 按钮）：
   - 标题：**阿里云 / cn-shanghai**（**vllm** / **ecs.gn6i-c4g1.xlarge**）
   - **计算资源**：**GPU** `NVIDIA T4 x 1` / **CPU / 内存** `4C / 15G`
   - **费用估算**：时估 `¥12.78/小时` / 日估 `¥306.72/天` / 月估 `¥9201.60/月` / 年估 `¥111952.80/年`
6. 确认推理引擎、业务策略与算力方案后，点击 **"确认部署"** 按钮 → 弹出 **Step 3 "发布部署"** 配置弹窗，进入部署任务配置。

### Step 3：发布部署（弹窗配置）

1. 在 Step 2 点击 **"确认部署"** 按钮后，弹出 **"发布部署"** 配置弹窗（覆盖在智能推荐部署页之上，可关闭）。
2. 弹窗顶部蓝色信息条显示 **"已选模型：Qwen3-8b (Qwen3 / 文本生成 / 1.0.0 / 128)"** 与右上角 **"部署模式"** 提示（与 Step 2 选择一致）。
3. 配置必填项：
   - **"*云账号"**（下拉选择，必填）：从已接入的云账号中选择（如 `aliyun-test`）。
   - **"*任务名称"**（文本输入，必填，字符计数 0/36）：输入部署任务名称（如 `qwen3_8b_vllm_std`，当前 17/36）。
4. 配置可选项：
   - **"任务描述"**（多行文本，选填，字符计数 0/200）：输入任务说明（如"可选说明"）。
5. 选择操作：
   - 点击 **"返回推荐"** 按钮 → 关闭弹窗，返回 Step 2 智能推荐部署页（可调整算力方案）。
   - 点击 **"发起部署"** 按钮 → 提交部署任务，部署完成后可在 **"我的部署"** 列表中查看状态。

![Step 3：发布部署](./images/step-3-publish-deployment.png)

| 返回推荐 | 按钮 | — | 选填，关闭弹窗返回 Step 2 调整算力方案 |

## 其他操作

| 操作名称 | 操作步骤 |
|----------|----------|
| 刷新资源 | 在 Step 1 模型库页面，点击右上角的 **"刷新资源"** 按钮 → 重新拉取可部署的云账号与模型列表 |
| 查看模型详情 | 在 Step 1 模型库页面，点击目标模型行的 **"详情"** 按钮 → 查看模型的完整介绍、核心特性、技术规格 |
| 调整算力方案 | 在 Step 3 弹窗中，点击 **"返回推荐"** 按钮 → 返回 Step 2 智能推荐部署页，重新选择推理引擎 / 业务策略 / 算力方案 |
| 发起部署 | 在 Step 3 弹窗中，确认必填项已填写（云账号 + 任务名称）后，点击 **"发起部署"** 按钮 → 提交部署任务 |

## 注意事项

- **3 步流程顺序**：快速部署为 **3 步流程**（Step 1 选择云账号与模型 → Step 2 智能推荐部署 → Step 3 发布部署弹窗），前一步未完成时无法进入下一步。
- **Step 2 ≠ 提交部署**：Step 2 的 **"确认部署"** 按钮仅打开 Step 3 弹窗，并不直接提交；真正的提交在 Step 3 的 **"发起部署"** 按钮。
- **必填项校验**：Step 3 弹窗的 **"发起部署"** 按钮仅在 **"云账号"** 与 **"任务名称"** 均填写后启用；任一为空则按钮置灰。
- **字符上限**：任务名称上限 36 字符，任务描述上限 200 字符；输入框右下角实时显示字符计数。
- **已选模型不可改**：Step 3 弹窗的 **"已选模型"** 信息卡（顶部）来自 Step 1 的选择，不可在 Step 3 中修改；如需更换模型，点击弹窗外部区域关闭后，返回 Step 1 重新选择。
- **算力方案跟随选择**：Step 2 选择不同的 **推理引擎**（vLLM / SGLang）和 **业务策略**（经济适用 / 卓越性能 / 现货速达 / 显卡数量）会影响 Step 3 弹窗的 **"已选方案"**（如 `阿里云 / cn-shanghai / vllm / ecs.gn6i-c4g1.xlarge`）；调整方案后弹窗信息会同步更新。
- **前置账号接入**：使用快速部署前，需先在 **"接入管理 > 接入账号"** 页面添加目标云平台的账号凭证（Access Key ID / Secret），否则 Step 3 的"云账号"下拉无可选项。
