# 我的部署

## 前言

| 项目   | 内容                                       |
| ---- | ---------------------------------------- |
| 适用角色 | 普通用户                                     |
| 导航路径 | 模型服务 > 我的部署                              |
| 功能定位 | 用户查看与管理已部署模型实例的入口，支持状态查询、详情查看（4 个子页签：基本信息 / API调用 / 监控信息 / 事件记录）、启停控制、发布与删除等操作 |

## 页面结构

"我的部署"由 2 大区块组成：① 部署任务列表（带筛选与 Tab）→ ② 部署详情页（4 个子页签）。

### 部署任务列表

- 顶部为 **6 个云平台 Tab**（全部 / 华为云 / 亚马逊 / 阿里云 / 谷歌云 / AGIOne-powerone[Private]），默认选中"全部"
- 下方为 **三联筛选**（名称 / 状态 / 模型名称），含"搜索"（紫色）与"重置"按钮
- 主区为 **部署任务卡片列表**，每张卡片含：部署名称 + 状态徽章（● 运行中 / ● 启动中 / ● 已停止 等）+ 模型信息（模型名 / 推理引擎 · 版本）+ 部署元数据（部署模式 / 所属云平台 / 地域）+ 资源规格（GPU × N / N 核 / N G）+ 当前单价（¥/小时）+ 创建时间
- 卡片底部 4 个直接操作：**"启动"** / **"停止"** / **"详情"** / **"..."**（更多）
- 卡片右上角 **"..."**（更多）菜单含 4 个子项：**发布** / **删除** / 监控信息 / API调用详情
- 底部为分页控件（共 N 条 / N 条/页 / ‹ / 1 / › + 前往 N 页 输入框）

### 部署详情页

点击列表中任意部署卡片的 **"详情"** 按钮 → 进入详情页，页面顶部显示部署名称（如 `qwen3_8b_vllm_std`），左上角返回箭头返回列表。详情页含 4 个子页签：

- **基本信息**（默认，紫色高亮）— 4 张卡片：**基本信息**（4×2 字段网格）+ **访问方式**（带状态徽章 + Endpoint + API Key + 复制 按钮）+ **部署信息**（**部署策略** + **资源配置** 2 个子区）+ **模型信息**（模型名 + 元模型 ID / 模型类型 / 模型标签）
- **API调用** — 顶部小字 `aliyun  区域: cn-shanghai` 标识 + **请求URL** / **请求方法** POST / **请求头** 表（Content-Type / Authorization，5 列：参数 / 参数名 / 默认值 / 参数类型 / 是否必须）+ **请求参数** 表（同 5 列）+ **代码样例**（7 个语言 Tab：curl / python / java / go / nodejs / php / ruby + 复制 按钮）
- **监控信息** — 2 组 Tab（**监控维度**：基础资源指标 / 服务性能/业务指标 / GPU专项监控 + **监控周期**：1分钟 / 15分钟 / 1小时 / 1天）+ 12 张折线图（基础资源指标 6 张 + 服务性能/业务指标 2 张 + GPU专项监控 4 张）
- **事件记录** — 3 列表格（**事件类型** / **事件信息** JSON / **首次发生时间** ISO 8601）

## 操作步骤

### 查询与筛选部署任务

1. 进入平台首页，点击左侧导航栏的 **"模型服务 > 我的部署"** 菜单，进入我的部署页面。
2. 页面顶部默认展示 **"全部"** Tab（紫色高亮），其余 5 个云平台 Tab 依次为：**华为云** / **亚马逊** / **阿里云** / **谷歌云** / **AGIOne-powerone**（含 **Private** 标签）。
3. 点击目标云平台 Tab（如 **阿里云**），系统将仅展示该云平台下的部署任务。
4. 在搜索栏使用三联筛选：
   - **"名称"** 文本框：输入部署任务名称的关键字；
   - **"状态"** 下拉选择：按部署状态筛选；
   - **"模型名称"** 文本框：输入已部署模型名称的关键字。
5. 点击紫色 **"搜索"** 按钮应用筛选条件；点击 **"重置"** 按钮清空所有筛选条件。
6. 列表为空时显示空状态插画 + **"暂无数据"** 提示。

#### 参数说明 - 列表筛选

| 字段名称 | 字段类型 | 示例 | 说明 |
|----------|----------|------|------|
| 云平台 Tab | 单选 | `全部` / `华为云` / `亚马逊` / `阿里云` / `谷歌云` / `AGIOne-powerone`（Private） | 必填，按云平台筛选 |
| 名称 | 文本 | `qwen3_8b_vllm_std` | 选填，部署任务名称关键字 |
| 状态 | 下拉选择 | `运行中` / `启动中` | 选填，按部署状态筛选 |
| 模型名称 | 文本 | `Qwen3-8b` | 选填，已部署模型名称关键字 |
| 页码 | 数值 | `1` | 必填，当前页码（**"前往"** 输入框可跳转） |

### 查看部署详情s

在「我的部署」列表中点击目标部署卡片的 **"详情"** 链接 → 进入部署详情页。

#### 基本信息（默认显示）

页面分 4 张卡片展示：

- **基本信息** 卡片：
   - **部署名称**（如 `qwen3_8b_vllm_std`）/ **服务模型**（如 `Qwen3-8b`）/ **部署模式**（如 `Custom` / `Single Node`）/ **云厂商**（如 `阿里云`）
   - **当前状态**（如 `● 运行中` 绿色 / `● 启动中`）/ **创建时间**（如 `2026-06-24 15:32:46`）/ **最后更新时间**（如 `2026-06-24 15:37:28`）/ **地域**（如 `cn-shanghai`）

- **访问方式** 卡片：
   - **服务状态**：绿色 **服务可用** 徽章（部署完成后显示）
   - **Endpoint**（如 `http://1780814767926817.cn-shanghai.pai-eas.aliyuncs.com/api/predict/qwen3_8b_vllm_std`）+ 复制 按钮
   - **API Key**（如 `MTc5MTliZGQ5MDZlYTFlZWFiNmFiNmNlOTE2ODBhZTdlNWNlZmY2Yg==`）+ 复制 按钮

- **部署信息** 卡片（含 **"部署策略"** 与 **"资源配置"** 两个子区）：
   - **部署策略**：**业务策略**（如 `Balanced` / `经济适用`）/ **推理框架类型**（如 `vllm` / `sglang`）/ **预估每计费周期成本**（如 `¥11.10 / 小时 × 1 实例`）
   - **资源配置**：**云账号**（如 `aliyun-test`）/ **云厂商**（如 `阿里云`）/ **地域**（如 `cn-shanghai`）/ **规格 ID**（如 `ecs.gn7i-c16g1.4xlarge`）/ **实例数量**（如 `1`）/ **GPU**（如 `NVIDIA A10 x1`）/ **CPU**（如 `16核`）/ **内存**（如 `60G`）/ **当前单价**（如 `¥11.10/小时`）/ **云侧资源 ID**（如 `eas-m-j7qim87p6ja63kgeg7`）

- **模型信息** 卡片：模型名 / **元模型 ID**（如 `qwen/qwen3-8b`）/ **模型类型** / **模型标签**

#### API调用

顶部小字 **"aliyun  区域: cn-shanghai"**（云账号与地域标识）。页面分 4 段：

- **请求URL**：部署完成后自动生成（部署中时为占位符，访问方式卡片有"自动生成访问地址"的提示）
- **请求方法**：`POST`
- **请求头** 表格（5 列：参数 / 参数名 / 默认值 / 参数类型 / 是否必须）：

  | 参数 | 参数名 | 默认值 | 参数类型 | 是否必须 |
  | --- | --- | --- | --- | --- |
  | Content-Type | Content-Type | `application/json` | string | true |
  | Authorization | Authorization | `Bearer {api_key}` | string | true |

- **请求参数** 表格：

  | 参数 | 参数名 | 默认值 | 参数类型 | 是否必须 |
  | --- | --- | --- | --- | --- |
  | max_tokens | max_tokens | `1024` | string | true |
  | messages | messages | `[{"role":"user","content":"hello"}]` | string | true |

- **代码样例** 区（7 个语言 Tab + 右上角 复制 按钮）：curl / python / java / go / nodejs / php / ruby

![API调用子页签](./images/api-access.png)

#### 监控信息
- **监控维度** Tab（3 选 1）：**基础资源指标**（默认）/ 服务性能/业务指标 / GPU专项监控
- **监控周期** Tab（4 选 1）：**1分钟**（默认）/ 15分钟 / 1小时 / 1天
- **折线图** 网格（按维度动态切换）：
  - **基础资源指标**（6 张）：CPU使用率（%）/ CPU核心使用量（core）/ 内存利用率（%）/ 内存使用量（MiB）/ 入方向流量（count/s）/ 出方向流量（count/s）
  - **服务性能/业务指标**（2 张）：每秒查询次数（count/s）/ 平均响应时间（μs）
  - **GPU专项监控**（4 张）：GPU利用率（%）/ GPU显存利用率（%）/ GPU显存总量（MiB）/ GPU显存使用量（MiB）

> 注：部署刚启动时（截图所示），12 张折线图均为空（无数据），需等监控数据采集完成后填充。

### 启停部署

1. 在「我的部署」列表中，找到目标部署卡片。
2. 点击卡片底部的 **"启动"** / **"停止"** 链接 → 启动/停止部署任务（部署卡片在 `运行中` 状态下显示 `停止` 链接，在 `启动中` / `已停止` 状态下显示 `启动` 链接）。

### 发布部署

1. 在「我的部署」列表中，找到目标部署卡片。
2. 点击卡片底部的 **"..."**（更多）按钮 → 弹出 4 项操作菜单（**发布** / **删除**  / 监控信息 / API调用详情）。
3. 选择 **"发布"** → 弹出 **"发布模型"** 弹窗（**"选择发布区域"** 标题 + 描述 "这个模型要发布到哪个区域？发布后，调用方将通过统一的平台 API 访问。"）。
4. 在弹窗中选择发布位置：
   - **私有区**（左侧卡片，🔒 图标）：发布到私有模型库，**仅当前组织内**可见与调用（标签：**组织内调用** / **不进入公共目录** / **发布到私有区** →）。
   - **公共区**（右侧卡片，紫色高亮选中）：发布到公共模型库，**对平台用户开放调用**（标签：**进入平台公共目录** / **可独立定价与计费** / **发布到公共区** →）。
5. 点击底部 **"发布到私有区"** / **"发布到公共区"** 按钮 → 完成发布；如放弃，点击弹窗右上角 **"×"** 关闭按钮或弹窗外部区域关闭。

## 注意事项

- **部署流程入口**：当前部署操作已统一迁移至 **"模型服务 > 快速部署"** 菜单（Step 1 选择云账号与模型 → Step 2 智能推荐部署 → Step 3 发布部署弹窗）。**"我的部署"** 页面仅用于查看与管理已创建的部署任务，不再提供创建部署的入口。
- **筛选与重置**：使用三联筛选（**名称** / **状态** / **模型名称**）后，可点击 **"重置"** 按钮清空条件；使用 **"前往"** 输入页码可快速跳转。
- **详情页 4 个子页签**：
  - **基本信息**（默认）— 展示 **基本信息** + **访问方式** + **部署信息** + **模型信息** 4 张卡片
  - **API调用** — 展示 **请求URL** / **请求方法** / **请求头** / **请求参数** + 7 种语言 **代码样例**
  - **监控信息** — 展示 3 个监控维度（**基础资源指标** / **服务性能/业务指标** / **GPU专项监控**）× 4 个时间窗口的 12 张实时指标折线图
