# Enable Cloud Resource Pools

Enable the cloud regions that AGIOne may use and maintain readable localized names.

## Procedure

1. Open `Access Management > Resource Pools`.
2. Select the cloud platform and review discovered regions.
3. Enable only regions approved for the target business.
4. Maintain localized display names and verify region state.

See [Resource Pools](../../../../usermanual/ai-infra-on-cloud/operator/access-management/resource-pools/).

## Completion Checklist

- [ ] Required regions are enabled and clearly named.
- [ ] Unapproved regions remain unavailable.
- [ ] Authorization pages can select the enabled regions.
