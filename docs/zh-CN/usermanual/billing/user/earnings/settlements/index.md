# 月度结算单

::: info 文档信息
版本：v1.0
更新日期：2026-07-10
:::

## 功能概述

月度结算单用于按账期查看 Provider 收益的结算记录，包括账期、状态、应结算、实际到账、抹零、到账时间和操作入口。服务商可在该页面筛选结算单、查看详情，并在具备权限时处理转入相关操作。

| 项目 | 内容 |
| --- | --- |
| 适用角色 | 服务商账号、服务商财务查看人员、收益结算处理人员 |
| 菜单路径 | Provider 收益 > 月度结算单 |
| 页面用途 | 查看月度结算记录、核对结算状态、进入结算详情 |
| 关键入口 | 搜索、重置、转入、详情 |

### 新手理解

月度结算单像服务商收益的月结单，记录收益从统计到到账的账期过程。每一行代表一个账期，状态用于判断该账期处于进行中、待结算还是已结算。查看金额前应先确认账期，处理转入前应先核对结算状态和到账信息。

### 术语速查

| 术语 | 含义 | 处理建议 |
| --- | --- | --- |
| 月度结算单 | Provider 某账期收益结算记录。 | 按账期查看状态和金额。 |
| 应结算 | 按规则计算出的应结算金额。 | 与收益总览和明细核对。 |
| 实际到账 | 实际进入目标账户的金额。 | 与到账时间和出账流水一起核对。 |
| 抹零 | 结算过程中因规则产生的舍入或调整金额。 | 比较应结算和到账时必须考虑。 |
| 到账时间 | 结算金额完成到账的时间。 | 状态处理中时不要提前下结论。 |
| 转入 | 将满足条件的结算金额转入可用余额的入口。 | 操作前确认账期、金额和权限。 |

## 前提条件

1. 当前账号具备查看 Provider 收益结算单的权限。
2. 已进入 `Provider 收益 > 月度结算单`。
3. 执行转入类操作前，已完成账期、金额和权限复核。

## 主要操作

### 查询结算单

1. 进入 `Provider 收益 > 月度结算单`。
2. 在 `全部状态` 中选择需要查看的结算状态。
3. 在 `账期` 中选择目标账期。
4. 点击 `搜索` 查看筛选结果。
5. 如需恢复默认列表，点击 `重置`。

### 查看结算单详情

1. 在结算单列表中定位目标账期。
2. 查看该行的状态、应结算、实际到账、抹零和到账时间。
3. 点击行内 `详情`。
4. 在详情页或详情区域核对结算构成、到账信息和处理状态。

### 处理转入入口

1. 在列表中确认目标账期状态。
2. 点击行内 `转入`。
3. 在弹窗或页面提示中核对账期、金额和到账账户。
4. 涉及资金状态变化时，应由具备权限的人员复核后再继续。
5. 如账期、金额或状态不符合预期，取消操作并返回列表核对。

## 踩坑提示

- 应结算和实际到账不一致时，先看抹零、到账时间和状态，不要直接判定异常。
- 当前账期仍在处理中时，结算金额和到账信息可能继续变化。
- 转入操作可能影响收益账户和可用余额，确认账期、金额和权限前不要继续。
- 查询不到账期不一定是无收益，也可能是状态筛选过窄或结算单尚未生成。

## 常见问题

### 查询不到目标账期

**问题现象：**

选择账期后列表为空，或没有看到预期结算单。

**可能原因：**

筛选状态与账期不匹配，或该账期还没有生成可见结算记录。

**处理方式：**

点击 `重置` 后重新选择账期；如仍为空，返回收益总览确认该账期是否已有收益和结算提示。

### 应结算与实际到账不一致

**问题现象：**

某账期的应结算金额与实际到账金额不同。

**可能原因：**

结算过程中存在抹零、状态处理中，或到账信息尚未完成更新。

**处理方式：**

查看该行 `抹零`、`到账时间` 和 `详情` 信息，必要时结合出账流水核对。

### 转入按钮无法继续

**问题现象：**

点击 `转入` 后无法继续，或页面提示当前结算单不能转入。

**可能原因：**

当前账期状态不满足转入条件，或账号缺少处理权限。

**处理方式：**

核对结算单状态和账号权限；未满足条件时不要绕过页面提示继续处理。

## 注意事项

- 月度结算单涉及收益和到账信息，不要在外部渠道传播未脱敏截图。
- 转入、结算确认等操作可能改变资金状态，必须先核对账期和金额。
- 结算状态与到账时间可能存在处理延迟，核对时应结合详情和出账流水。
