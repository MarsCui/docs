# Quick Deployment

Deploy a cloud model by selecting an authorized account, model, inference engine, strategy, and compute plan.

## Procedure

1. Open the cloud model marketplace and start quick deployment.
2. Select the access account, authorized region, and model.
3. Review the recommended deployment mode, inference engine, business strategy, and compute plan.
4. Confirm cost and capacity and submit the deployment.
5. Open My Deployments and watch state and events.

See [Quick Deployment](../../../../usermanual/ai-infra-on-cloud/user/model-services/quick-deployment/).

## Completion Checklist

- [ ] Account, region, model, engine, and compute plan are selectable.
- [ ] The request is submitted with the expected cost and scope.
- [ ] My Deployments contains the new record.
