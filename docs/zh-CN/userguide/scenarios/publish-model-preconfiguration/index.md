# 场景概览-发布模型（预设置）

指导运营管理员在 Provider 发布公有模型前，完成平台侧的基础数据与审批配置。

## 适用角色

- 运营管理员

## 场景目标

- 准备模型作者、元模型、模型来源和模型模板。
- 准备标签、币种和模型审核能力。
- 让模型提供方能够基于统一配置发布模型。

## 开始前准备

- 已明确模型类型、输入输出模态和协议。
- 已准备上游模型 Endpoint、地域和请求头规则。
- 已明确计费币种和审核责任人。

## 完成检查

- [ ] Provider 能选择目标作者、元模型、模型来源和模板。
- [ ] 模态、协议、Token 限制和高级能力配置一致。
- [ ] 标签和币种满足发布与计费需要。
- [ ] 模型审核入口和审批角色可用。

## 常见失败分支

| 现象 | 优先检查 |
| --- | --- |
| Provider 看不到模板 | 模板状态、发布范围、元模型和模型来源关联 |
| 模型来源测试失败 | 地域节点、请求头、协议地址和连通性 |
| 计费配置无法完成 | 币种、计费单位、模型类型和价格规则 |
