# 场景概览-充值与计费

本场景帮助用户区分充值记录、资源配额、资源用量和模型计费，并核对额度是否到账、资源是否可用以及消耗为何产生。

## 适用角色

- 最终用户、模型提供方、运营管理员

## 场景目标

- 确认充值记录和账户额度变化。
- 区分资源配额不足与账户额度不足。
- 将扣减记录对应到实例、作业或模型调用。

## 开始前准备

- 明确要核对模型调用还是 On-Prem 资源使用。
- 明确租户、账号、时间范围、账期和计费单位。

## 推荐阅读路径

1. 查看充值记录
2. 查看资源配额和账户额度
3. 查看资源或模型用量
4. 按账期核对扣减明细
