# 接入账号

## 前言

| 项目   | 内容                |
| ---- | ----------------- |
| 适用角色 | 普通用户 |
| 导航路径 | 接入管理 > 接入账号       |
| 功能定位 | 提供云平台账号接入、编辑和删除功能 |

## 页面结构

### 搜索区域

页面顶部提供云平台筛选栏和搜索框，以及 **"搜索"** 和 **"重置"** 按钮。

### 操作按钮区

页面右上角提供 **"添加云账号"** 按钮，用于新增云平台账号。

### 数据列表说明

账号卡片列表展示已接入的云平台账号，显示账号名称、云平台、账号类型、创建时间等信息。

## 操作步骤

### 添加云账号

1. 进入平台首页，点击左侧导航栏的 **"接入管理"** 菜单，进入接入账号管理页面。
2. 点击页面右上角的 **"添加云账号"** 按钮，弹出「新增账号」窗口。

![接入账号](./images/access-accounts-list.png)

3. 配置账号信息：
   - 填写 **账号名称**（如 `aliyun-test`）；
   - 选择 **云平台**（如阿里云、华为云、亚马逊等）；
   - 填写 **Access Key ID**；
   - 填写 **Access Key Secret**。
4. 确认所有信息配置无误后，点击 **"确定"** 按钮完成添加。

![添加账号](./images/add-account.png)

#### 参数说明

| 字段名称 | 字段类型 | 示例 | 说明 |
|----------|----------|------|------|
| 账号名称 | 文本 | `aliyun-test` | 必填，自定义账号标识 |
| 选择云平台 | 下拉选择 | `阿里云` | 必填，选择账号所属的云平台 |
| Access Key ID | 文本 | `your-access-key-id` | 必填，云平台的访问密钥 ID |
| Access Key Secret | 文本 | `your-access-key-secret` | 必填，云平台的访问密钥 Secret |

## 其他操作

| 操作名称 | 操作步骤 |
|----------|----------|
| 编辑账号 | 点击目标账号卡片右上角的 **"..."**（更多）按钮 → 选择 **"编辑"** → 修改账号信息 → 点击 **"确定"** |
| 删除账号 | 点击目标账号卡片右上角的 **"..."**（更多）按钮 → 选择 **"删除"** → 确认操作（**删除后数据将无法恢复，请谨慎操作**） |

## 注意事项

- **Access Key Secret** 为敏感信息，请妥善保管，切勿泄露。
- **删除账号操作不可逆**，删除后数据将无法恢复，请谨慎操作。
- 确保填写的 Access Key ID 和 Access Key Secret 与云平台控制台一致，否则可能导致资源拉取失败。
