# 通用问题

## 账号相关

### 问：无法登录，优先检查什么？

- 校验账号密码或验证码是否有效
- 确认账号未被锁定/禁用
- 如启用企业登录，确认 SSO 绑定是否完成

### 问：如何重置密码？

- 通过“忘记密码”（如已启用）自助重置，或联系管理员在管理门户重置。

## 部署相关

### 问：安装时拉取镜像失败怎么办？

- 确认节点到镜像仓库网络可达
- 校验仓库凭证与证书信任链
- 检查 DNS 与时间同步（NTP）

### 问：是否通过界面进行部署？

答：是。模型部署可以通过界面完成，方式包括使用运营方管理的模型模板启动 Kubernetes Job 或 Deployment，或提供指定命令。

### 问：是否也可以通过 CLI 部署？

答：不可以。部署通过平台提供的界面完成，因此无需通过 CLI 执行。

## 使用相关

### 问：模型请求变慢，如何排查？

- 查看 GPU/CPU/内存利用率是否异常
- 检查请求日志与错误率
- 核对并发与配额策略是否合理

## 开发工作台

### 问：平台提供什么样的开发工作台方案？

答：平台提供基于 Web 的 IDE，包括 Jupyter 和 CodeServer。

### 问：平台是否为开发者提供 Jupyter 或 CodeServer 等 Web IDE？

答：提供 Jupyter 和 CodeServer。
