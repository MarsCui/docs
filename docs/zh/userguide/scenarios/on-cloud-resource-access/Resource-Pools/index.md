# 接入资源池

## 操作步骤

### 启用 / 禁用资源池地域

1. 进入平台首页，点击左侧导航栏的 **"接入资源池"** 菜单，进入资源池管理页面。
2. 在左侧云平台列表上方输入关键字搜索云厂商（包含 阿里云 26 个 / 谷歌云 41 个 / 华为云 24 个 / aws 亚马逊 34 个 / AGIOne-powerone 1 个 等），点击目标云平台（如 阿里云）。
3. 右侧展示选中云平台的所有地域卡片网格（每行 3 列），地域包括 华北1(青岛)、华北2(北京)、华北3(张家口) 等。
4. 找到需要启用 / 禁用的地域，点击地域卡片右侧的开关按钮：
   - 开关为开启状态时，表示该地域已启用；
   - 点击开关即可切换地域的启用 / 禁用状态。
![接入资源池](Resource-Pools-list.png)

### 编辑地域显示名称

1. 进入平台首页，点击左侧导航栏的 **"接入资源池"** 菜单，进入资源池管理页面。
2. 在左侧选择目标云厂商（如 阿里云），找到需要编辑的地域。
3. 点击地域卡片上的编辑图标，弹出「编辑名称」窗口。
4. **"显示名称"**（标注"多语言"）：用于设置地域在列表、详情和选择控件中的显示名称。点击 **"English"** / **"中文简体"** 标签切换语言 Tab，**"将在 English 语言环境下显示"** / **"将在中文简体语言环境下显示"** 的提示，分别在对应 Tab 下填写英文与中文简体环境下的名称。
5. 确认配置无误后，点击 **"确定"** 按钮完成修改；如需放弃，点击 **"取消"**。
![编辑名称](./images/Edit-Name.png)
#### 参数说明

| 字段名称 | 字段类型 | 示例 | 说明 |
|----------|----------|------|------|
| 云厂商 | 选择框 | `阿里云` / `亚马逊` | 必填，选择要管理的云厂商资源池 |
| 地域开关 | 开关 | `开启` / `关闭` | 用于启用或禁用对应地域的资源池 |
| 显示名称 | 多语言文本 | English: `China (Beijing)` / 中文简体: `北京` | 必填，分别在"English"和"中文简体"Tab 下配置显示名称 |
