## 功能概述

`在线IDE` 用于创建和管理交互式开发环境。普通用户可选择单节点或集群形态，基于平台镜像和规格启动开发环境，用于代码调试、数据处理和实验验证。

| 项目 | 内容 |
| --- | --- |
| 适用角色 | 普通用户 |
| 导航路径 | AI基础设施 > On-Prem > 开发资源 > 在线IDE |
| 页面路由 | `/powerone/dev-training/dev-ide` |
| 管理对象 | 在线开发环境、单节点 IDE、集群 IDE、镜像、规格和运行状态 |
| 典型途径 | 创建交互式开发环境，在线调试代码、运行 Notebook 或准备训练和推理脚本 |

#### 新手理解

在线 IDE 像运行在资源池里的云端开发机。你不需要本地安装 GPU 环境，只要选择镜像和规格，平台会创建一个可访问的开发环境。

#### 首次使用流程

1. 进入 `开发资源 > 在线IDE`。
2. 查看现有 IDE 列表和状态。
3. 点击 `创建 IDE（Create IDE）`。
4. 选择 `单节点（Single Node）` 或 `集群（Cluster）` 部署类型。
5. 继续填写镜像、规格、存储和启动参数。
6. 创建后返回列表查看状态并进入 IDE。

#### 术语速查

| 术语 | 说明 |
| --- | --- |
| 镜像 | 运行作业所需的容器环境，通常来自平台镜像服务或自定义镜像项目。 |
| 规格 | 作业可申请的资源套餐，例如 CPU、内存、GPU 型号和卡数。 |
| 配额 | 租户可使用的资源上限，常见维度包括 GPU、CPU、内存和规格。 |
| 单节点（Single Node） | 单节点开发环境，适合常规调试和小规模实验。 |
| 集群（Cluster） | 集群开发环境，适合需要主从节点或分布式资源的场景。 |

## 前提条件

1. 当前账号具备创建在线 IDE 的权限。
2. 目标地域有可用规格和配额。
3. 已有可用镜像，且镜像中包含所需开发工具。
4. 如需挂载数据或代码目录，相关存储已准备。

## 主要操作

### 创建在线 IDE

#### 适用场景

当需要一个可交互的开发、调试或 Notebook 环境时，创建在线 IDE。

#### 操作前确认

1. 已确认目标规格和配额可用。
2. 已确认镜像包含 Python、CUDA、框架或其他开发依赖。
3. 已确认运行周期，避免长时间空转消耗资源。

#### 操作步骤

1. 进入 `AI Infra > On-Prem > 开发资源 > 在线IDE`。
2. 点击 `创建 IDE`。
3. 在部署类型页面选择 `单节点` 或 `集群`。

![选择部署类型](./images/deployment-type.png)

4. 点击 `填写参数`，进入在线 IDE 创建配置页面。
5. 按页面字段查看或填写 IDE 名称、地域、镜像、规格、存储挂载和启动命令等配置。
6. 核对配置项、资源规格、存储路径和运行方式。
7. 如仅学习或截图，只查看字段和页面状态，不点击最终 `提交`、`确定` 或 `确认`。

![创建在线 IDE](./images/create-ide.png)

## 参数说明

| 字段名称 | 是否必填 | 字段类型 | 说明 |
| --- | --- | --- | --- |
| IDE 名称 | 必填 | 文本 | 在线 IDE 展示名称。 |
| 部署类型 | 必填 | 单选 | 选择单节点或集群形态。 |
| 地域 | 必填 | 下拉选择 | 选择创建在线 IDE 的目标地域。 |
| 镜像 | 必填 | 下拉选择 | 选择开发环境镜像。 |
| 资源规格 | 必填 | 下拉选择 | 选择在线 IDE 使用的算力规格。 |
| 存储挂载 | 否 | 路径 | 配置代码、数据或输出目录的挂载路径。 |
| 启动命令 | 否 | 文本 | 配置在线 IDE 启动或运行命令。 |
## 踩坑提示

- 集群模式可能需要更多配额，普通调试优先使用单节点。
- 镜像缺少依赖时，IDE 能启动但代码可能运行失败。
- `提交`、`确定`、`确认` 属于最终动作。
- 创建在线 IDE 会占用配额和集群资源。
- 学习或截图时只查看页面、字段、弹窗和状态，不提交真实创建任务。
- 不写真实租户、地域、镜像地址、规格 ID、存储路径、Token、密码、Endpoint、启动参数或测试数据。

## 配置规则与影响

- 在线 IDE 会占用配额和集群资源，运行时间越长消耗越多。
- 镜像、规格和存储由运营方配置，普通用户只能选择可见项。
- 停止、删除或释放 IDE 前，应确认代码和输出已保存到持久化目录。

## 常见问题

#### 创建 IDE 按钮不可用或看不到

**问题现象：**列表页没有创建入口。

**可能原因：**

- 当前账号没有创建权限。
- 所选地域未开放在线 IDE。
- 页面加载或权限状态异常。

**处理方式：**

1. 刷新页面。
2. 确认当前地域。
3. 联系运营方检查账号权限和资源池配置。

#### IDE 创建后无法打开

**问题现象：**列表有 IDE，但访问入口打不开或页面一直加载。

**可能原因：**

- 实例仍在启动。
- 镜像启动脚本异常。
- 网络入口或代理不可用。

**处理方式：**

1. 等待状态进入运行中。
2. 查看实例日志或事件。
3. 联系运营方检查集群入口和镜像配置。

## 后续操作

1. 进入 IDE 编写和调试代码。
2. 将重要代码、数据和输出保存到持久化存储。
3. 不用时停止或释放 IDE。

## 注意事项

- 不要把账号密码、Token、私钥、Endpoint、启动参数或测试数据写入 Notebook、代码仓库或截图。
- 创建前确认存储挂载路径，避免输出只保存在临时容器目录。
- 学习或截图时只查看页面、字段、弹窗和状态，不提交真实创建任务。
