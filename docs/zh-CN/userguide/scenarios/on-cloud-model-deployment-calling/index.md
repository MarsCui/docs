# 场景概览-On Cloud 模型部署与调用

指导用户（租户成员）从云上模型广场快速部署模型实例，并获取 API 调用能力。

## 适用角色

- 模型提供方、最终用户

## 场景目标

- 接入或确认可用云账号。
- 选择目标模型、推理引擎和算力方案完成部署。
- 从我的部署获取状态、监控和 API 调用能力。

## 开始前准备

- 运营管理员已完成云资源、业务地域和租户授权。
- 目标模型资产已上架。
- 用户已准备经过处理的云账号凭据样例。

## 推荐阅读路径

1. 接入账号
2. 快速部署
3. 在我的部署检查状态、事件、监控和 API

## 完成检查

- [ ] 用户接入账号校验成功并能看到被授权地域。
- [ ] 快速部署能够选择目标模型、推理引擎和算力方案。
- [ ] 部署状态进入运行中，事件和监控没有持续错误。
- [ ] 使用脱敏后的 Endpoint 和 API Key 完成受控调用。
- [ ] 停止或删除测试部署后，状态和资源释放符合预期。

## 常见失败分支

| 现象 | 优先检查 |
| --- | --- |
| 无可用账号或地域 | 云账号校验、资源授权和租户范围 |
| 智能推荐没有方案 | 模型资产、框架、算力方案和地域容量 |
| 部署长期创建中或失败 | 事件、云账号权限、镜像、配额和云侧资源状态 |
| 部署运行但调用失败 | Endpoint、认证、端口、协议和服务健康 |
