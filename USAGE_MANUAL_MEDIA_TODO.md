# 使用手册视频/GIF 素材规划

本文记录高频操作页面后续可补充的视频或 GIF 素材规划，不写入发布手册正文。

## 待补素材

| 场景 | 建议素材 | 关键画面 | 脱敏要求 |
| --- | --- | --- | --- |
| 集群注册 | 30-60 秒 GIF | 进入集群管理、填写接入信息、提交并查看状态 | 遮挡 kubeconfig、API Server、token、节点 IP |
| On-Cloud 快速部署 | 45-90 秒 GIF | 选择模型、地域、规格、提交部署、查看我的部署 | 遮挡 Endpoint、API Key、云账号 ID、费用 |
| Playground 文本体验 | 30-60 秒 GIF | 选择模型、填写 Prompt、调整参数、查看流式输出 | 遮挡 Personal Key、客户 Prompt、请求 ID |

## 制作前检查

1. 使用中文界面重新录制。
2. 统一窗口尺寸和鼠标移动节奏。
3. 所有账号、密钥、Endpoint、客户名称、租户名称和内部资源编号必须脱敏。
4. 素材文件名使用小写英文和连字符。
