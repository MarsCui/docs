# 场景概览-注册与登录

本场景介绍创建账号、密码登录、邮箱验证码登录和找回密码，是进入 AGIOne 各子系统前的统一账号入口。

## 适用角色

- 最终用户、模型提供方、运营管理员

## 场景目标

- 完成账号注册或使用已有账号登录。
- 在忘记密码时通过邮箱验证恢复访问。
- 登录后进入账号权限对应的工作台。

## 开始前准备

- 可访问的企业邮箱。
- 已阅读平台隐私政策和服务条款。
- 已有账号时，准备用户名或邮箱及密码。

## 推荐阅读路径

1. 创建账号
2. 选择密码或邮箱验证码登录
3. 登录失败时执行密码找回
