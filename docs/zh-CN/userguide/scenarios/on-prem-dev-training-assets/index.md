# 场景概览-On-Prem 开发训练与资产沉淀

本场景介绍用户如何创建开发环境、提交训练任务，并把模型、镜像、数据集和输出结果沉淀为可复用资产。

## 适用角色

- 最终用户
- 负责镜像、存储和资源配置的运营管理员

## 场景目标

- 开发环境和训练任务能够使用已授权算力。
- 数据和输出写入持久存储。
- 训练产物能够进入后续模型部署流程。

## 开始前准备

- 已有可用镜像、资源规格和租户配额。
- 已准备代码、数据集和持久存储路径。

## 推荐阅读路径

1. 创建开发环境
2. 挂载代码、数据和存储
3. 提交训练任务
4. 保存模型和日志等产物
