# 图像生成

## 前言

| 项目 | 内容 |
|------|------|
| 适用角色 | 普通用户 |
| 导航路径 | 体验中心 > 图像生成 |
| 功能定位 | 通过文本描述生成 AI 图像，体验模型的图像生成能力 |

## 页面结构

### 搜索区域

主页面无搜索区域；「选择模型」弹窗左侧提供模型名称 / 模型标识搜索框。

### 操作按钮区

* 顶部模型下拉框用于查看当前图像模型并打开「选择模型」弹窗
* 右上角提供「多个模型对比」和「返回旧版」按钮
* 输入框左下角提供 Size、张数、Thinking mode 和参数配置按钮
* 输入框右下角提供调用密钥选择和发送按钮

### 数据列表说明

页面中央展示图像生成提示区、提示词输入框、参数配置入口和生成结果区域。

## 操作步骤

### 模型生成图像

1. 进入平台首页，点击左侧导航栏的 **"体验中心 > 图像生成"** 菜单，进入图像生成体验页面。
2. 点击顶部模型下拉框，打开「选择模型」弹窗：
   - 可在左侧搜索框输入模型名称或模型标识进行筛选；
   - 在左侧模型列表中选择目标图像模型；
   - 在右侧供应方区域选择可用供应方实例；
   - 点击「确定」完成模型切换。

![选择模型](./images/select-model.png)

3. 在输入框左下角设置基础生成选项：
   - 选择 **Size**，设置生成图片尺寸；
   - 设置 **张数**，控制单次生成图片数量；
   - 按需开启或关闭 **Thinking mode**。
4. 点击参数配置按钮，设置高级参数：
   - 选择 **协议**（如 openai/images）；
   - 按需开启或关闭 **Enable sequential**；
   - 按需开启或关闭 **Watermark**；
   - 填写必填的 **Seed**。

![图像生成参数配置](./images/parameters.png)

5. 如需指定调用凭证，在输入框右下角选择对应的 **Personal Key**。
6. 在输入框中描述想生成的图片，点击发送按钮生成图像。
7. 生成完成后，在页面结果区域查看生成的图片。

#### 参数说明（参数配置面板）

| 字段名称 | 字段类型 | 示例 | 说明 |
|----------|----------|------|------|
| Size | 快捷选项 | `2K` | 生成图片的尺寸规格 |
| 张数 | 数值选项 | `1` | 单次请求生成的图片数量 |
| Thinking mode | 开关 | `开启` | 是否启用思考模式 |
| 协议 | 下拉选择 | `openai/images` | 模型调用的 API 协议 |
| Enable sequential | 开关 | `关闭` | 是否按顺序生成多张图片 |
| Watermark | 开关 | `关闭` | 是否为生成图片添加水印 |
| Seed | 数值输入 | `0` | 必填，生成随机种子，用于控制生成结果的可复现性 |
| Personal Key | 下拉选择 | `Personal Key 20260616...` | 当前请求使用的调用凭证 |
| 提示词 | 文本输入 | `请描述你想生成的图片` | 输入框中的图片生成描述 |

#### 参数说明（模型选择弹窗）

| 字段名称 | 字段类型 | 示例 | 说明 |
|----------|----------|------|------|
| 模型名称 / 标识 | 文本 | `Mock Ali Wan 2.7 Image / mock-models/wan2.7-image` | 模型的名称与唯一标识 |
| 发布日期 | 日期 | `2026-07-01` | 模型的发布时间 |
| 上下文长度 | 文本 | `-` | 图像模型的上下文长度展示值 |
| 输入 / 输出 Credit | 数值 | `- Credit / 2 Credit` | 模型输入和输出侧的计费展示 |
| 供应方 | 文本 | `Model Mocker:Ali Wan 2.7 Image Mocker / AGIOneSystem` | 模型的供应方及服务实例 |
| 计费单位 | 文本 | `Per Image` | 当前供应方实例的计费方式 |
| 图片价格 | 数值 | `2 Credits/Image` | 单张图片的调用费用 |
| 延迟 / 吞吐量 / 成功率 | 数值 | `- / 0 t/s / -` | 供应方实例的运行指标 |
| 周调用量 / 月图片量 | 数值 | `0 / 0 images` | 该供应方实例的近期使用情况 |
| 状态标签 | 标签 | `推荐 / 已上架` | 模型或供应方实例的推荐和上架状态 |
| 能力标签 | 标签 | `深度思考 / 工具调用` | 模型能力标签；截图中均为不支持状态 |

## 其他操作

| 操作名称 | 操作步骤 |
|----------|----------|
| 切换模型 | 点击顶部模型下拉框 → 在弹窗中选择不同模型或供应方 → 点击「确定」 |
| 搜索模型 | 在「选择模型」弹窗左侧搜索框输入模型名称或模型标识，快速定位目标模型 |
| 调整基础参数 | 在输入框左下角修改 Size、张数或 Thinking mode |
| 调整高级参数 | 点击输入框左下角参数配置按钮 → 修改协议、Enable sequential、Watermark、Seed 等参数 |
| 选择调用密钥 | 点击输入框右下角密钥下拉框，选择本次请求使用的 Personal Key |
| 多个模型对比 | 点击「多个模型对比」按钮，进入多模型并行图像生成体验页面 |
| 文本输入 | 在底部输入框输入图片描述，按发送按钮或 Enter 发送；使用 Shift+Enter 换行 |
| 返回旧版 | 点击「返回旧版」按钮，切换到旧版图像生成页面 |

## 注意事项

* 图片描述越清晰，生成结果通常越接近预期。
* Size 和张数会影响生成耗时和计费，请按实际需要设置。
* Seed 为必填项，相同 Seed 在相同模型和参数下有助于复现相近结果。
* Watermark 是否可用取决于所选协议、模型和供应方实例能力。
* 模型价格、状态、计费单位和性能指标以「选择模型」弹窗中展示的信息为准。
* 可点击「多个模型对比」按钮进入多模型并行图像生成体验页面。
