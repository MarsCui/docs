# 场景概览-模型发布审批

本场景介绍运营管理员如何审核模型提供方提交的发布申请，核对模型信息、协议、计费、限流和发布范围。

## 适用角色

- 运营管理员
- 等待审核结果的模型提供方

## 场景目标

- 审核项能够回溯到发布配置。
- 通过或驳回都有明确结论和原因。
- 审批后的模型可见范围符合发布设置。

## 开始前准备

- 已有待审核的模型发布申请。
- 已明确模型发布规范、计费规则和可见范围。

## 推荐阅读路径

1. 定位待审核申请
2. 核对模型和协议
3. 核对计费、限流和发布范围
4. 通过或填写可执行的驳回意见
