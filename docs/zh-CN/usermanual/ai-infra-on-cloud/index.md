::: info 文档信息
版本：v1.0
更新日期：2026-07-08
:::

## 子系统定位

多平台调度面向云上资源接入、授权、资源池治理和云上模型服务部署，连接云厂商账号、资源池、业务授权、部署资产和调度策略。

#### 新手理解

多平台调度像多云资源调度枢纽：运营管理员把不同云厂商的账号和资源接入平台，按租户和业务地域授权，普通用户再基于被授权的资源快速部署模型服务。

## 核心术语速查

| 术语 | 说明 |
| --- | --- |
| 云账号 | 接入云厂商资源的账号或授权主体。 |
| 资源池 | 从云账号中抽象出来、可授权和调度的资源集合。 |
| 租户授权 | 控制租户能使用哪些云账号、资源池和地域。 |
| 业务地域授权 | 控制业务场景可以部署到哪些地域或资源范围。 |
| 部署资产 | 模型库、推理框架、运行镜像和部署配置。 |
| 调度策略 | 决定云上部署如何选择资源、回退和优先级。 |

## 推荐阅读路径

1. 新手先读 [多平台调度快速入门](./getting-started/) 理解云账号、资源池和授权关系。
2. 运营管理员按云平台、云账号、资源池、授权、部署资产、调度策略顺序配置。
3. 普通用户从接入账号开始确认可用资源，再发起快速部署；部署提交后，可在 `模型及AI服务 > 创作空间 > 我的部署` 的 `On-Cloud` 列表查看并继续发布。
4. 需要完整流程时阅读 [从零开始部署云上模型服务](./end-to-end/deploy-cloud-model-service/)。

## 使用前提

1. 运营管理员已准备云厂商授权和最小权限策略。
2. 资源池、租户授权、业务地域授权和部署资产已配置。
3. 普通用户已获得目标账号、地域、资源池和模型部署权限。
4. 涉及部署和调用前，已确认 Endpoint、API Key、费用和资源容量。

## 踩坑提示

- 云账号接入成功不代表用户可部署，还必须完成资源池启用、租户授权和业务地域授权。
- 快速部署没有可用资源时，不要只改模型参数，应先核对授权范围、资源池状态和库存。
- 我的部署显示创建中时，先看事件记录和资源推荐结果，再判断是否需要运营管理员介入。
- 快速部署、发起部署、启停部署、发布部署、选择发布区域、发布模型、删除、禁用、解绑和授权调整属于高风险动作，执行前必须确认影响范围和回退方案。
- 学习或截图场景只查看菜单、页面、字段、状态和跳转，不提交真实部署或配置变更。
- 不记录真实 AK/SK、API Key、Token、Endpoint、账号 ID、部署 ID、客户信息、费用明细、内部地址或测试参数。

## 常见问题

#### 用户为什么看不到云资源

**问题现象：**

普通用户在快速部署或我的接入账号中看不到目标账号、地域或资源。

**可能原因：**

- 云账号未完成校验或资源同步。
- 租户授权或业务地域授权未覆盖当前账号。
- 资源池未启用或容量不足。

**处理方式：**

1. 先查看我的接入账号中的授权状态。
2. 联系运营管理员核对资源池、租户和业务地域授权。
3. 资源池异常时等待同步或切换可用地域。

#### 部署为什么创建失败

**问题现象：**

快速部署提交后任务失败、长时间创建中或无法生成调用信息。

**可能原因：**

- 云账号授权失效或资源池不可用。
- 模型、框架、镜像或调度策略不匹配。
- 费用、配额或实例容量不足。

**处理方式：**

1. 查看我的部署中的事件、状态和错误信息。
2. 核对模型、规格、地域和授权账号是否一致。
3. 提供脱敏部署 ID、时间和错误摘要给运营管理员。

## 后续操作

1. 运营管理员定期检查云账号密钥轮换、资源同步和授权范围。
2. 普通用户部署后查看 Endpoint、API Key、事件和监控；如需继续发布，进入 `模型及AI服务 > 创作空间 > 我的部署` 的 `On-Cloud` 列表选择发布区域，并跳转到 [我的模型](../model-services/user/studio/my-models/) 的发布模型页面。
3. 费用或容量异常时，结合多平台调度部署记录、模型及AI服务调用记录和云厂商账单核对。

## 注意事项

- 多平台调度入口页只帮助判断云账号、授权、资源池和部署入口，不替代具体云资源配置说明。
- 云账号、授权链路和资源池范围必须统一，否则用户看到的部署选项和费用口径会不同。
- 涉及云账号凭据、角色授权、资源池变更、部署删除、禁用、解绑、启停部署或授权调整时，应先确认影响租户、地域和业务范围。
- 截图、工单和排障记录中不要暴露真实 AK/SK、API Key、Token、Endpoint、账号 ID、部署 ID、客户信息、费用明细、内部地址或测试参数。
