## 文档目的

[支持矩阵](./support-matrix)用于判断产品范围属于支持、条件支持、规划中还是暂不支持。本页说明即使某项能力为“支持”，仍可能阻塞实际流程的条件。

## “支持”不等于“不需要前提条件”

| 范围 | 常见限制 | 必须完成的动作 |
| --- | --- | --- |
| 账号与访问 | 正确菜单可能因角色、租户或授权范围不可见 | 排查功能前先确认任务归属和资源授权 |
| 云上资源 | 平台类型存在，不代表凭据、API、地域、账号或资源池可用 | 验证准确接入方式、账号权限、网络和资源清单 |
| 加速卡 | 清单内加速卡仍可能与驱动、运行时、镜像、推理引擎或模型不兼容 | 验证完整的加速卡-软件-模型组合 |
| 模型 | 模型条目或 Endpoint 存在，不代表所需模态、上下文、usage 字段、可用性或性能满足要求 | 测试准确模型、Endpoint、请求、响应、配额和目标负载 |
| 部署 | 支持某种安装方式，不代表当前主机、端口、存储和离线资产已就绪 | 执行文档预检并解决全部阻塞项 |
| 运营数据 | 监控、调用、用量、计量和收益可能具有不同范围或同步延迟 | 确认角色范围、时间范围、采集状态和源记录 |
| 财务与 License | 余额、结算、对账、收益和 License 状态可能使用不同账期、组织、账户或同步状态 | 作出财务结论前确认账期、组织、客户、账户、License 范围和源记录 |
| 设置与流控 | 成员、角色、登录、Key、审计、平台设置或 API 流控变更可能影响真实用户和流量 | 修改设置前确认角色范围、组织范围、审批依据、发布状态和回退方式 |

## 访问与流程限制

- 菜单和数据会随角色、租户与授权范围变化。
- 运营方必须先准备资源和基础配置，模型提供方或普通用户才能完成后续任务。
- 模型提供方发布单模型或聚合模型；普通用户使用已授权模型，不创建聚合模型。
- 即使页面可访问，审核状态、可见范围、配额、额度和资源状态仍可能阻塞发布、部署或调用。
- 凭据、部署、发布、审核、充值和付费调用操作需要获得批准，并按安全要求处理。
- 财务和 License 页面需要统一账期、组织、客户和账户口径，不应跨不同范围直接比较余额、结算、收益或 License 额度。
- 设置、登录策略、Key、角色、审计记录和 API 流控规则可能影响在线用户或流量，编辑、删除、发布、重置和导出都应作为高风险操作处理。

## 云平台限制

- 华为云接入暂不支持。
- 云平台管理能力不代表支持任意云厂商 API、地域、服务类型、实例规格或账号策略。
- 云账号权限、网络路由、API 可用性、凭据、地域资源、资源池和授权必须一起验证。
- 停用或修改云平台可能影响引用它的账号、资源池、部署资产和用户部署。

## 加速卡与模型限制

- 加速卡纳管与模型兼容属于两项独立验证。
- 混合 CPU 架构、混合加速卡类型或未验证驱动栈可能影响调度和运行稳定性。
- 显存、卡数、并行策略、量化方式、上下文长度、存储带宽和网络拓扑共同决定模型能否运行并达到目标性能。
- 多模态模型行为取决于准确模型、预处理、Endpoint、请求格式和运行资产。
- 聚合模型需要符合条件的成员模型和部署版本可用的路由策略，验收时需考虑成员模型之间的差异。
- RAG 和 Function Calling 为规划中能力，不属于当前生产能力。

## 安装与基础设施限制

- 单节点安装不等同于高可用生产架构。
- 多节点和算力节点安装需要满足安装指南规定的节点数量、角色、私网 IP、SSH、时间同步、端口、存储和运行环境要求。
- 已有 Docker、Kubernetes、运行数据或端口占用可能阻塞安装。未获得授权且没有回滚方案时，不得清理现有环境。
- 离线交付依赖完整的安装包、镜像、运行时、清单和校验文件。
- 网络端口与方向应遵循[网络规划](../technical/network)和安装指南，向任意来源开放端口不能替代网络设计。
- 容量、高可用、备份、恢复、恢复目标和性能指标需要按项目设计并测试。

## 交付前检查清单

- [ ] 目标能力在[支持矩阵](./support-matrix)中为“支持”，或已明确接受“条件支持”。
- [ ] 目标角色、租户、菜单和资源授权已确认。
- [ ] 目标云平台或加速卡组合已验证。
- [ ] 准确的模型、Endpoint、镜像、驱动、运行时、存储和网络组合已测试。
- [ ] 配额、额度、定价、计量和数据可见范围已确认。
- [ ] 涉及财务或设置时，已确认账期、客户/组织范围、结算源记录、License 状态以及设置/API 流控影响。
- [ ] 安装预检和必要网络测试通过。
- [ ] 容量、稳定性、故障处理、备份和恢复验收标准已记录。
- [ ] 暂不支持和规划中能力未写入交付承诺。
