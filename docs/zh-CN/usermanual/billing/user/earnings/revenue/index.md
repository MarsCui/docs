# 收益总览

::: info 文档信息
版本：v1.0
更新日期：2026-07-10
:::

## 功能概述

收益总览用于查看服务商侧 Provider 收益的整体情况，包括收益账户余额、当前自然月预估、下次结算提示、历史结算表现、本月新客户、在用模型 / 应用数量，以及按账期统计的日收益、客户贡献、模型贡献和结算单概览。

| 项目 | 内容 |
| --- | --- |
| 适用角色 | 服务商账号、服务商财务查看人员、收益运营人员 |
| 菜单路径 | Provider 收益 > 收益总览 |
| 页面用途 | 查看收益总览、进入出账流水、结算单和客户明细 |
| 关键入口 | 出账流水、结算单、客户明细、转入可用余额 |

### 新手理解

收益总览像服务商收益看板。先通过顶部指标判断收益账户当前状态，再通过账期分析查看某个账期内的收益趋势、主要客户和主要模型来源。需要核对明细时，再进入出账流水、月度结算单或客户总览页面。

### 术语速查

| 术语 | 含义 | 处理建议 |
| --- | --- | --- |
| 收益账户余额 | Provider 当前可查看的收益账户余额。 | 转入前先与结算单和出账流水核对。 |
| 当前自然月预估 | 当前自然月内根据已有数据估算的收益。 | 不要当作最终结算金额。 |
| 账期分析 | 按指定账期展示日收益、客户贡献和模型贡献。 | 比较数据前先统一账期。 |
| 出账流水 | 收益账户发生出账或转入变化的明细记录。 | 金额异常时优先核对。 |
| 转入可用余额 | 将收益账户金额转入可用余额的处理入口。 | 操作前复核账期、金额和权限。 |

## 前提条件

1. 当前账号具备查看 Provider 收益的权限。
2. 已进入 `Provider 收益 > 收益总览`。
3. 页面数据加载完成后再核对金额、账期和趋势图。

## 主要操作

### 查看收益概览

1. 进入 `Provider 收益 > 收益总览`。
2. 查看顶部收益账户余额、当前自然月预估和下次结算提示。
3. 继续查看历史结算、本月新客户、在用模型 / 应用等指标。
4. 如需核对趋势，查看下方 `账期分析` 区域。

### 切换账期分析

1. 在 `账期分析` 区域选择账期。
2. 查看日收益图表，判断收益在账期内的变化趋势。
3. 查看 Top 5 客户和 Top 5 模型，定位主要收益来源。
4. 查看结算单概览，确认当前账期对应的结算状态。

### 进入关联明细

1. 点击 `出账流水`，进入收益账户出账流水页面核对出账记录。
2. 点击 `结算单`，进入月度结算单页面查看账期结算记录。
3. 点击 `客户明细`，进入客户总览页面查看客户维度收益。
4. 返回收益总览后，可继续按账期或指标进行核对。

### 转入可用余额

1. 在收益总览页面点击 `转入可用余额`。
2. 在弹窗或页面提示中核对账期、金额和目标账户。
3. 涉及余额转入时，应由具备权限的人员复核后再继续。
4. 若信息不一致，取消操作并先核对结算单、出账流水和客户明细。

## 踩坑提示

- 不要只看收益账户余额判断收益是否正确，应结合出账流水、月度结算单和客户明细核对来源。
- 当前自然月预估会随消费和结算处理变化，不要当作最终到账金额。
- Top 客户或 Top 模型为空不一定是异常，先确认账期内是否有真实收益。
- 点击转入可用余额前，必须确认账期、金额、目标账户和当前账号权限。

## 常见问题

### 收益余额与预期不一致

**问题现象：**

收益总览中的余额或预估收益与内部记录不一致。

**可能原因：**

所选账期不同、出账流水尚未全部核对，或结算单状态仍在处理中。

**处理方式：**

先确认账期，再进入 `出账流水`、`结算单` 和 `客户明细` 分别核对来源、状态和客户维度数据。

### Top 客户或 Top 模型为空

**问题现象：**

账期分析中客户排行或模型排行没有数据。

**可能原因：**

当前账期内没有对应收益，或页面数据仍在加载。

**处理方式：**

切换到已有收益的账期重新查看；如果其他指标也为空，检查账号权限和页面加载状态。

### 转入可用余额前需要检查什么

**问题现象：**

点击 `转入可用余额` 后出现需要继续确认的操作入口。

**可能原因：**

该操作会影响收益账户和可用余额，需要确认账期、金额和权限。

**处理方式：**

先核对月度结算单和出账流水，再由具备权限的人员按页面提示继续处理。

## 后续操作

1. 需要核对结算状态时，进入 [月度结算单](../settlements/)。
2. 需要按客户维度核对收益时，进入 [客户总览](../customers/)。
3. 需要处理余额转入时，先完成账期和金额复核。

## 注意事项

- 收益总览展示的是汇总视图，最终核对应结合出账流水、结算单和客户明细。
- 涉及转入可用余额、结算确认或客户敏感信息查看时，应先确认操作权限。
- 不要将包含客户名称、金额、账号或联系方式的截图直接用于外部沟通。
