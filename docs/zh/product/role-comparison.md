# AGIOne 平台 — 角色对比总览

admin · operator · provider · eu（终端用户）— 角色边界与核心能力一览

---

## 角色速览

| 角色 | 定位 | 商城类比 |
|--------|------|----------|
| **admin** — 平台管理员 | 平台级系统元数据治理者 | 平台管理层（证照资质） |
| **operator** — 平台运营 | 平台标准与审核治理者 | 商城运营团队（规则与审核） |
| **provider** — 供应商 / 提供方 | AI 能力生产与部署方 | 品牌 / 供应商（货品上架） |
| **eu** — 终端用户 | AI 能力使用者 | 消费者（购买使用） |

---

## admin — 平台管理员

**一句话定位**：平台级系统元数据治理者

**典型身份**：平台所有者、IT 负责人、系统管理员

**业务域**：用户空间 · License

### 核心能力

- ✓ 创建 / 冻结租户
- ✓ 创建用户账号
- ✓ 分配 / 解绑角色
- ✓ 配置菜单与权限
- ✓ 定义角色权限模板
- ✓ 管理系统内置角色

### 不在范围内

- ✗ 使用或体验模型
- ✗ 发布或审核模型
- ✗ 部署算力任务
- ✗ 处理账务与充值

### 关键入口

- **用户空间 → 系统管理**：租户管理 · 用户管理 · 角色管理 · 菜单管理 · 内置角色管理
- **License → 许可证管理**：许可证发放 · 激活 · 到期续期

### 典型场景

- 新客户入驻 → 激活租户 + 创建并分配用户账号
- 新功能上线 → 新增菜单入口，绑定到对应角色
- 平台许可证到期 → License 管理 → 续期激活，保障平台功能正常

---

## operator — 平台运营

**一句话定位**：平台标准与审核治理者

**典型身份**：运营经理、产品运营、商务运营

**业务域**：模创 · 算枢 · 算模方 · 账务

> **域说明**
> - **模创**：模型创建 · 发布 · 审核 · 应用开发
> - **算枢**：云平台接入 · 模型部署 · 推理
> - **算模方**：异构算力 · 训练 · 数据集 · 镜像
> - **账务**：账务运营 · 账户 · 充值 · 协议

### 核心能力

- ✓ 审核模型 / 应用发布
- ✓ 管理模型分类体系
- ✓ 接入云平台（阿里 / 华为 / 腾讯 / 天翼）
- ✓ 定义算力规格与镜像
- ✓ 配置推理路由策略
- ✓ 管理供应商准入规则
- ✓ 查看客户充值记录
- ✓ 发布平台公告

### 不在范围内

- ✗ 管理租户与用户元数据
- ✗ 直接调用或使用模型
- ✗ 部署个人算力任务
- ✗ 查看其他供应商数据

### 关键入口

- **模创 → 运营管理**：模型 / 应用审核
- **算枢 → 接入管理**：云平台接入 / 资源池授权
- **算模方 → 资源配置**：规格管理 / 镜像管理
- **账务 → 运营管理**：公告管理 / 协议管理

### 典型场景

- 供应商提交大模型 → 按平台标准审核，批准或驳回
- 接入华为云 → 五步接入指南，配置 AK/SK，连接资源池
- 高峰期响应慢 → 调整推理路由策略（多副本负载均衡），验证效果

---

## provider — 供应商 / 提供方

**一句话定位**：AI 能力生产与部署方

**典型身份**：AI 开发者、供应商商务、企业 IT 负责人

**业务域**：模创 · 算枢 · 算模方

### 核心能力

- ✓ 创建 / 发布模型
- ✓ 开发 AI 应用
- ✓ 一键部署模型
- ✓ 创建训练 / 推理任务
- ✓ 上传数据集
- ✓ 构建私有镜像
- ✓ 查看用量统计与收益
- ✓ 管理发布状态（草稿 / 私有 / 公开）

### 不在范围内

- ✗ 审核他人的模型 / 应用
- ✗ 管理平台级配置（标签 / 策略 / 规格）
- ✗ 查看其他供应商的数据
- ✗ 直接公开发布（需审核）

### 关键入口

- **模创 → 工作空间**：我的模型 / 我的应用
- **算枢 → 模型服务**：快速开始 / 我的部署
- **算模方 → 任务中心**：模型训练 / 模型推理
- **算模方 → 创意工坊**：工作空间 / 我的创作

### 典型场景

- 提供 AI API → 创建模型 → 申请公开发布 → 等待运营审核 → 上线
- 部署图像生成模型 → 算枢市场 → 一键部署 → 获取 API 端点供内部系统接入
- 训练行业模型 → 选择数据集 + GPU 规格 + 参数 → 系统自动调度

---

## eu — 终端用户

**一句话定位**：AI 能力使用者

**典型身份**：业务人员、业务用户、API 开发者

**业务域**：模创 · 账务 · 算枢（个人）

### 核心能力

- ✓ 浏览模型 / 应用市场
- ✓ 在线体验
- ✓ 调用 API（需 API Key）
- ✓ 创建聚合模型工作流
- ✓ 查看调用记录与分析
- ✓ 个人账户充值
- ✓ 个人模型部署（需授权）
- ✓ 收藏模型与应用

### 不在范围内

- ✗ 创建并发布模型
- ✗ 审核他人的模型 / 应用
- ✗ 管理平台级配置
- ✗ 查看他人的用量数据

### 关键入口

- **模创 → 发现**：模型市场 / 应用市场
- **模创 → 体验**：文本 / 图片 / 视频 / 音频
- **模创 → 我的调用**：调用分析 / 调用记录
- **账务 → 我的账务**：充值 / 消费记录

### 典型场景

- 业务人员撰写文案 → 模型市场 → 在线体验 → 获取 API Key 接入工作流
- 集成图像识别 → 阅读 API 文档 → 调用图像模型 → 接入 CRM 系统
- 串联多个模型 → 在我的聚合模型中拖拽连接 → 生成组合 API
- 余额不足 → 我的账务 → 充值 → 继续使用
