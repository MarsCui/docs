# 场景概览-模型消费与收益

本场景介绍调用方如何核对模型用量，模型提供方如何核对客户调用和收益，以及双方如何按相同账期完成对账。

## 适用角色

- 最终用户、模型提供方

## 场景目标

- 用量数据能够对应到调用日志。
- 收益数据能够回溯到有效调用和价格规则。
- 异常能够定位到模型、调用方、账期或计费配置。

## 开始前准备

- 明确模型、时间范围、账期和计费单位。
- 准备脱敏后的请求时间或调用标识。

## 推荐阅读路径

1. 查看调用日志
2. 查看模型用量
3. 提供方查看模型收益
4. 对照计费配置完成核对

- 调用、用量和收益在同一时间范围内可以互相验证。
