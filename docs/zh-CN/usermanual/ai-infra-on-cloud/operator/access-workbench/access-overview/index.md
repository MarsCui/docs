## 功能概述

`接入总览` 用于维护接入账号、资源池、授权状态、部署资产和异常提示，支撑多云调度、资源授权和模型部署流程。

| 项目 | 内容 |
| --- | --- |
| 适用角色 | 运营方 |
| 导航路径 | 接入工作台 > 接入总览 |
| 页面路由 | /operator/access-workbench/access-overview |
| 管理对象 | 接入账号、资源池、授权状态、部署资产和异常提示 |
| 典型用途 | 总览多云接入进度和资源可用状态 |

### 新手理解

接入总览像多云接入的仪表盘。它不替代具体配置页，而是帮助运营方快速看到账号、资源池、授权和同步任务哪里正常、哪里需要处理。

### 术语速查

| 术语 | 说明 |
| --- | --- |
| 接入进度 | 云平台、云账号、资源池和授权的完成情况。 |
| 异常任务 | 账号校验、资源同步或授权同步中的失败项。 |
| 资源池水位 | 云资源池容量和使用率概览。 |
| 授权覆盖 | 资源已开放给租户或业务地域的范围。 |
## 前提条件

1. 至少已接入一个云平台或云账号。
2. 当前账号具备接入总览查看权限。
3. 资源池、授权和同步任务已按计划接入。
## 页面说明

页面用于汇总云平台接入进度、云账号状态、资源池容量、授权覆盖和异常任务。运营方可以从总览进入具体页面处理凭据失败、资源同步异常或授权缺口。

## 主要操作

### 操作步骤

1. 进入 `接入工作台 > 接入总览`。
2. 查看云平台、云账号、资源池和授权卡片。
3. 按异常状态定位失败账号、同步失败资源池或未授权地域。
4. 点击卡片或明细跳转到对应功能页处理。
5. 处理后返回总览确认状态恢复。

### 参数说明

| 字段名称 | 是否必填 | 字段类型 | 示例 | 说明 |
| --- | --- | --- | --- | --- |
| 统计范围 | 是 | 筛选项 | `全部云平台` | 总览卡片统计的云平台或地域范围。 |
| 云账号状态 | 系统生成 | 枚举 | `异常` | 账号凭据和连通性聚合状态。 |
| 资源池水位 | 系统生成 | 百分比 | `72%` | 资源使用和剩余容量概览。 |
| 授权覆盖 | 系统生成 | 数字 | `8 个租户` | 已完成授权的租户或业务地域数量。 |
| 异常任务 | 系统生成 | 列表 | `同步失败` | 需要运营方处理的任务类型。 |

### 踩坑提示

- 总览是聚合数据，排障时要跳转到具体账号、资源池或授权页核对。
- 统计数据可能有同步延迟，不适合替代云厂商实时控制台。
- 截图时遮挡租户名称、云账号名称和异常资源标识。

### 结果校验

1. 总览卡片展示账号、资源池、授权和异常统计。
2. 异常项可以跳转到对应详情页。
3. 处理异常后总览状态能在同步后更新。

## 常见问题

### 总览显示异常但详情页正常

**问题现象：**

总览卡片仍显示异常，进入详情页后记录已恢复。

**可能原因：**

- 总览聚合数据存在刷新延迟。
- 异常任务已处理但统计缓存未更新。
- 筛选范围包含其他异常对象。

**处理方式：**

1. 刷新总览并确认更新时间。
2. 清空筛选范围后重新查看。
3. 等待同步任务完成后复核。

### 总览没有资源池水位

**问题现象：**

资源池卡片为空或水位为 0。

**可能原因：**

- 资源池未同步。
- 云账号校验失败。
- 当前账号没有资源池查看权限。

**处理方式：**

1. 检查云账号和资源池同步状态。
2. 确认当前账号权限。
3. 进入资源池页面核对资源识别结果。

## 后续操作

1. 处理异常云账号。
2. 同步资源池容量。
3. 补齐租户和业务地域授权。

## 注意事项

- 总览数据可能存在同步延迟。
- 异常项需要跳转到具体账号、资源池或授权页处理。
- 截图前遮挡云账号、租户和资源标识。
