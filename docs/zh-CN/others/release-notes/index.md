# 版本发布说明

## 2026-06-30

### 发布内容

#### 重点更新

- 【功能】新增并完善用户空间能力，支持工作区概览、项目管理、个人 / 项目 Model API Key、成员额度、额度申请、操作记录、预算和模型白名单等自助管理能力。
- 【功能】强化 API Key 与 Quota 管理链路，支持个人 Key / 项目 Key 归因、网关认证、配额日志、过期时间、限额统计，并优化异常场景下的额度扣减准确性。
- 【功能】模型服务能力持续升级，支持缓存计费、阶梯计费、Web Search 计费，以及图片、视频、语音等多模态模型的用量解析和扣费处理。
- 【功能】体验中心文本、图片、视频、语音页面完成结构升级，优化模型选择、参数配置、模型对比、文件上传、Key 选择和新旧版切换体验。
- 【功能】模型发布、模型市场、我的模型、元模型配置、模型来源和模型审核流程增强，提升模型上架、配置、审核、发布和多语言维护效率。
- 【功能】本地算力平台资源纳管能力升级，覆盖资源池、集群、加速卡、规格、镜像、存储、部署模板、显存计算、模型服务、运行实例和在线 IDE。
- 【功能】多云调度平台推荐部署和模型迁移能力增强，支持按模型、框架、部署模式、云商和策略生成推荐方案，并补充硬件规格、价格、库存和多云适配信息。
- 【功能】账务链路重构，覆盖账户概览、月度账单、账户流水、充值、模型供应方收益和运营管理员结算，并优化消费来源归因和 Stripe 落账可靠性。
- 【功能】授权许可能力增强，支持发布前预检、模型发布控制、SaaS 订阅入口、顶栏健康状态、全局告警、AI 基础设施守卫和 SKU 使用量管控。
- 【功能】登录与账号体系增强，新增邮箱账号找回、邮箱登录、密码重试限制和 Google 登录自动注册租户能力。
- 【功能】系统管理与权限能力增强，支持新版菜单标签、服务配置国际化、API 流控、权限扫描、系统配置动态刷新和定时任务参数化管理。
- 【界面】全平台界面体验持续统一，优化顶栏、卡片、筛选区、表格、操作列、空态、状态标签、弹窗、MessageBox、AK 下拉框和暗色模式样式。
- 【性能】优化体验中心页面切换、模型市场查询、Quota 扣减、云规格库存判断、充值落账、API 流控统计、定时任务状态同步、日志上报和监控指标链路。
- 【性能】优化网关远程调用、动态配置刷新、日志上报和监控指标链路，提升启动稳定性、接口兼容性和运维可观测性。
- 【质量】修复模型服务、账务归因、授权许可拦截、菜单权限、定时任务、Google 登录、邮件模板、多语言、暗色模式和复杂表格展示等稳定性问题。

#### 设置

- 【功能】新增工作区概览、项目、我的 Key、成员额度、额度申请、操作日志等用户空间入口。
- 【功能】项目支持项目列表、详情、成员、用量、活动记录、设置、预算、预算预警、重置周期、超预算策略和模型白名单。
- 【功能】项目 Key 支持创建、查看、复制、禁用 / 启用、轮换、撤销、过期、限额和用量统计。
- 【功能】API Key 全链路能力增强，支持网关归因认证、密钥名称、过期时间、默认配额、配额日志、个人 / 项目密钥归因和旧版密钥模型兼容。
- 【功能】区分 Model API Key 与旧 System AK/SK，能力边界更清晰，模型调用与普通接口调用的认证方式分离。
- 【功能】成员额度支持成员额度查看、管理员管理、额度策略、模型白名单、预警阈值和重置周期。
- 【功能】额度申请支持申请、取消、审批、驳回和自动过期，同一成员同一时间只能保留一条待处理申请。
- 【功能】客户目录能力增强，支持运营组织目录、租户客户摘要、客户类型筛选、业务身份、管理员邮箱和自定义客户标签等。
- 【性能】Quota 扣减规则更稳定，失败请求、白名单不通过、余额不足、Key 状态异常、项目归档、预检失败等场景不会产生不合理扣减。
- 【界面】Key 展示、列表表头、操作按钮、空态和额度相关文案统一，额度统一使用 Quota 表达。

#### 模型服务

- 【功能】模型计费能力增强，支持缓存计费、阶梯计费、Web Search 计费，以及图片、视频、语音等多模态模型的用量解析。
- 【功能】视频、图片和 Web Search 场景支持更完善，免密登录场景下也可获取视频、图片模型价格。
- 【功能】体验中心的文本、图片、视频、语音页面支持选择有效 Model API Key，并区分个人 Key 和项目 Key。
- 【功能】体验中心文本、图片、视频、语音页面升级为更集中的上下结构，模型选择、生成类型、模型对比和输入区位置更清晰，并支持返回旧版 / 去到新版入口。
- 【功能】体验中心模型对比能力优化，第二个模型默认展示为待选择态，未选择时给出轻量提示，并支持选择相同模型进行对比。
- 【功能】体验中心常用参数前置到输入框底部，模型协议和高级参数收拢到参数弹窗，文本、图片、视频、语音按各自真实配置展示参数。
- 【功能】上传能力增强，图片图生图、图片编辑、视频图生视频等场景支持上传入口、批量上传、缩略图、删除、上传中和失败提示。
- 【功能】AK / API Key 选择区并入输入框区域，下拉项展示名称、脱敏 Key 和 Personal 标签，并优化遮挡、层叠和宽度不足问题。
- 【性能】体验中心跨文本、图片、视频、语音页面切换时会清理上一页面的子类型、模型列表、对话内容和新旧版状态，并增加接口返回保护，避免旧页面请求覆盖当前页面。
- 【功能】模型发布流程增强，支持快速部署、我的部署、发布入口、多实例发布节点选择和发布后的返回路径处理。
- 【功能】模型市场和我的模型支持二级类型、计费方式筛选、模型名称搜索、预约下架模型查看和异常数据过滤。
- 【功能】运营管理员元模型配置体验优化，作者栏支持拖拽宽度，模型版本表格、输入 / 输出模态和模型能力展示更紧凑，并提供返回旧版入口。
- 【功能】元模型新增 / 编辑页支持模型描述多语言 tab、缺失语言红点提示和高级能力配置统一展示。
- 【功能】模型来源列表由卡片式展示升级为表格展示，供应商名称采用名称 + code 主次结构，地域较多时通过 `+N` 浮层查看完整地域和 API 地址。
- 【功能】模型来源新增 / 编辑页支持供应商名称、地域名称多语言 tab 编辑，按当前语言展示输入框，减少地域配置行高度。
- 【功能】模型审核列表升级为表格展示，分页调整到表格底部，审核弹窗增加能力配置只读展示，避免审核态误编辑。
- 【功能】模型作者显示名称、模型标签、模型描述、供应商名称和地域名称等多语言字段统一采用 tab + 红点提示模式。
- 【功能】供应商配置增加 output config 模版能力，平台内部参数和归因字段不会错误透传给厂商。
- 【功能】回调任务查询支持通过 taskId 或 id 查询，并加强非本人任务的数据隔离。
- 【性能】用量统计和扣费链路优化，支持标准 usage、原始厂商 usage、失败不扣费和异步视频任务补扣费。
- 【性能】模型市场元模型列表优化查询范围、字段返回和排序逻辑，减少无效数据和大字段影响。
- 【界面】模型详情、模型图标、模型来源、价格、Key 选择区和模型发布概览展示更清晰。
- 【界面】模型选择弹窗、参数弹窗、AK 下拉框、对话气泡、模型对比区域和暗色模式输入框样式统一，关闭弹窗后页面可立即恢复点击。

#### 本地算力平台

- 【功能】资源池管理能力升级，地域 / 可用区支持分栏详情、筛选、拖拽排序、关联组件、可用区启停和专属地域租户授权。
- 【功能】集群管理支持卡片 / 表格视图、筛选分页、详情抽屉、资源情况、关联规格、存储配置和启停维护。
- 【功能】加速卡管理升级为按厂商分组的资源目录，支持纳管状态、规格指标绑定、新增、编辑、删除和导入导出。
- 【功能】规格指标、资源规格的资源配置页支持更完整的卡片化管理和导入导出。
- 【功能】镜像、镜像组件、对象存储、块存储、文件存储等资源配置页支持更完整的卡片化管理。
- 【功能】部署模板管理能力改版，模型配置现在需要基于元模型创建，框架配置和推理模板页面视觉优化，并提供显存计算功能，用于支持用户根据场景计算部署模板所需显存。
- 【功能】用户侧概览页新增部署模板快速部署能力，可按模型和加速卡厂商选择模型资源并进入部署流程。
- 【功能】用户侧部署模板升级为交互式部署，用户选择模型后可根据场景自定义模型服务端能力，并按需求自动推荐加速卡资源部署方案。
- 【功能】新增模型服务功能，负责统一纳管模型部署任务。
- 【功能】移除原“模型训练”和“容器实例”功能，推出功能更完整的“运行实例”作为替代。
- 【功能】在线 IDE 和运行实例功能页面视觉优化。
- 【功能】在线 IDE 和运行实例终端、日志功能升级，终端支持同时打开多个，详情页内切换 Tab 页面不会断开连接；日志功能支持用户自定义展示容器日志文件。
- 【功能】用户端部分页面默认展示全地域数据，并隐藏顶栏的地域过滤组件。
- 【界面】资源卡片、状态标签、进度条、容量展示、操作菜单和详情抽屉体验统一，便于查看资源占用和异常原因。

#### 多云调度平台

- 【功能】推荐部署流程增强，支持按模型、框架、部署模式、云商和策略生成推荐方案，并确认部署。
- 【功能】推荐方案返回结构化硬件规格、价格、计费周期、币种、业务策略和实时库存状态，部署前判断更直观。
- 【功能】模型迁移新增导出、预检和导入能力，支持模型、云配置、API 配置、模型源和框架依赖在环境间迁移。
- 【功能】模型广场和模型管理补充元模型、云商、地域、模型标签、模型云配置和元模型快照信息。
- 【功能】接入对象总览支持展示云平台、资源池、账号、资产、租户授权和业务资源授权等聚合数据。
- 【功能】多云适配增强，优化 AWS、Aliyun 等云厂商相关部署、规格和访问地址处理。
- 【性能】云规格同步和库存状态判断优化，减少售罄或不可用规格被误认为可部署的情况。
- 【界面】模型列表、部署详情、云商筛选、元模型标签和部署错误提示展示更清晰。

#### 账务

- 【功能】账务链路重构，覆盖账户概览、月度账单、账户流水、充值、模型供应方收益和运营管理员结算。
- 【功能】模型供应方结算闭环增强，支持候选列表、生成向导、结算详情、进度展示、配对记录、划转、重投和结算巡检。
- 【功能】模型供应方客户列表补充关键词检索、组织、公司、管理员、邮箱、电话等识别字段。
- 【功能】账户流水补充来源系统、主体名称、展示名称和交易上下文，便于用户识别账务来源。
- 【功能】月度账单快照支持历史账期查询、热重建、手动刷新、对账任务和客户本月消费快照刷新。
- 【功能】客户通过 AGIOne 直接使用 Stripe 充值时，发送充值确认邮件通知，并优化 Stripe 侧交易明细信息的可读性与完整性。
- 【功能】消费来源归因优化为模型、AI 基础设施、项目、成员、Project 等维度，Project Key 消费和成员消费统计更准确。
- 【性能】充值和落账可靠性增强，优化 Stripe 超时后成功、重复回调、迟到成功落账、赠送积分和余额初始化等场景。
- 【界面】账期、金额、净变动、来源排行、项目名称、成员名称和模型供应方收益来源展示更易读。
- 【界面】模型供应方侧隐藏不可读的内部 ID，减少客户详情和收益流水中的理解成本。

#### 授权许可

- 【功能】授权许可增加发布前预检、模型发布控制、SaaS 订阅入口配置、顶栏健康状态、全局告警和 AI 基础设施守卫。
- 【功能】授权许可使用量聚合与请求次数管控增强，支持按 SKU 统计使用量、发布前权限预检和受限动作拦截。
- 【功能】授权许可风险口径补充已安装 SKU 已过期等场景，发布和审核拦截提示更准确。
- 【界面】授权许可管理页、全局告警、授权许可购买入口（可配置、可禁用）、联系邮箱等场景完整覆盖受限状态提示与引导。

#### 登录与账号

- 【功能】新增邮箱账号找回能力，支持账号找回联系邮箱配置和邮件模板 productName 配置。
- 【功能】密码登录支持邮箱作为登录标识，并增加密码登录重试限制。
- 【功能】Google 登录接入授权链路、授权码流程、系统配置、配置接口、自动注册租户和前端入口。
- 【界面】登录、注册、找回密码页面体验优化，支持视频背景、暗色表单、语言渲染和按钮悬停态。
- 【修复】修复注册后跳转、余额初始化重试、登录失败提示和邮件暗色模式渲染等问题。

#### 系统管理与权限

- 【功能】菜单权限 AppId 规则调整，菜单和角色授权继续使用前端 AppId，Permissions 弹窗使用后端 AppId。
- 【功能】新版系统菜单管理支持菜单适用角色标签，菜单标签支持保存、复制、筛选、排序和侧栏刷新。
- 【功能】服务配置国际化能力增强，完善英文资源、多前端应用菜单权限 AppId 语义和菜单删除应用 ID 参数。
- 【功能】新增 API 流控能力，支持网关级限流、本地计数、IP / 用户 / 租户 / OAuth2 身份条件、动态配额、规则排行、命中日志、统计看板和手动流控检查入口。
- 【性能】API 流控模型优化为计数键模型，支持节点变化后自动重算规则额度，并保留统计上报失败批次，便于排查。
- 【功能】权限扫描支持手动触发全部服务权限入库，并通过 MQ 触发本地扫描，限制主节点执行。
- 【功能】系统配置动态刷新和配置权限控制增强，缺省远程客户端启动行为更稳定。
- 【功能】定时任务支持参数元数据、参数指引、Cron 编辑器和 JSON 参数对象保存。
- 【性能】定时任务调度状态同步增强，支持暂停状态重启保持、运行中更新限制和执行日志倒序展示。
- 【功能】系统配置 JSON 编辑体验优化，币种接口权限和项目密钥权限标识完成调整。
- 【修复】修复菜单筛选区按钮挤压、菜单权限刷新、定时任务暂停状态、运行中更新和暂停后重启恢复等问题。

#### 工具与公共体验

- 【功能】显存计算和部署资源估算能力增强，可结合模型、框架、加速卡和参数估算部署所需显存、卡数和规格。
- 【功能】配置导入导出能力扩展到加速卡、规格指标、资源规格、模型配置、框架配置、推理模板和显存计算配置。
- 【界面】顶栏、通用卡片、状态下拉、筛选区、表格表头、操作列和空态样式统一。
- 【界面】模型白名单、Key 掩码、额度提示、项目预算、服务连接复制、部署入口等交互细节优化。
