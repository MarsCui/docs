## 2026-05-15

### 更新详情

#### 总体

- 【界面】全新UI，支持深色/浅色两种模式（包括全新首页、登录/注册页、Console页等）
- 【界面】全平台命名调整，平台统一了模型提供方和模型来源相关叫法，用户在不同页面看到的名称会更一致、更容易理解
- 【界面】发布模型概览 UI 优化，发布模型的概览页面做了展示优化，用户可以更清楚地查看模型发布前后的关键信息

#### 模型服务

- 【功能】模型调用支持Failover，模型调用自动切换能力，减少调用失败
- 【性能】模型调用支持熔断，优化调用监控和成功率判断，提升平台选择可用模型时的稳定性
- 【性能】改进流式输出，既支持统一化格式，同时支持模型原API接口能力，降低回答中断、格式异常、结果不完整等问题出现的概率
- 【功能】增强视频模型计费支持，token计费（每百万tokens）和非token计费（每second）场景
- 【界面】优化计费记录中的模型名称展示，账单和转账记录中可展示更准确、更易识别的模型信息
- 【性能】改进计费快照记录，便于后续核对使用量和费用
- 【性能】优化模型连通性测试接口，提高发布前检测的准确性
- 【功能】增加模型来源类型标识，便于区分不同来源的模型
- 【性能】增加平台数据同步（旁路同步），提升调用数据、计费数据、模型指标等统计数据的准确性
- 【性能】优化仪表板聚合统计，支持更稳定的小时级、天级数据汇总

#### 算枢模块

- 【功能】优化 AWS 相关模型注册、调用和部署流程
- 【界面】优化云账号查询体验，增加阿里云、华为云等简称识别
- 【功能】支持将云端推理服务 Endpoint 发布到模创平台
- 【功能】支持将推理服务 Token 发布到模创平台
- 【功能】支持将元模型信息同步到模创平台
- 【功能】支持在模创平台上对已发布模型进行推理调用测试

#### 账务

- 【功能】推进账务功能整合，将部分 MSP 侧账务能力合并到 operator 侧
- 【界面】余额展示优化，增加导航栏右上角展示

#### 工具

- 【功能】模型部署资源估算，通过模型、加速卡等参数估算部署该模型的显存和卡数

### 修复

- 【修复】改进模型参数异常时的处理方式，减少因参数不规范导致的调用失败
- 【修复】部署失败时的错误提示更详细，便于用户和运维人员定位问题
- 【修复】修复部分云厂商模型详情、规格查询和依赖兼容问题
- 【修复】改进历史数据刷新能力，减少大时间范围查询时超时或数据不完整的风险
- 【修复】修复支付配置默认值问题，提升账务配置稳定性
- 【修复】修正部分云区域权限查询逻辑，让可用区域展示更准确
- 【修复】修复英文文案中的特殊字符问题，提升多语言展示一致性
