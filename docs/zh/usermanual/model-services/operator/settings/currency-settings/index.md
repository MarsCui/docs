# 币种设置

## 前言

| 项目 | 内容 |
|------|------|
| 适用角色 | Operator |
| 导航路径 | 设置 > 币种设置 |
| 功能定位 | 配置平台统一使用的结算币种，全平台所有交易和费用显示均以此币种为准 |

## 页面结构

### 搜索区域

无搜索区域，直接展示当前币种配置。

### 操作按钮区

点击当前使用的币种卡片即可切换币种。

### 数据列表说明

页面展示当前平台统一使用的币种信息，包括币种代码和货币符号。

![Currency Settings](./images/currency-settings.png)

## 操作步骤

### 查看当前货币配置

1. 进入平台首页，点击左侧导航栏的 **"设置 > 币种设置"** 菜单，进入币种设置页面。
2. 查看当前平台统一使用的币种信息，页面提示"系统采用单一币种模式，设置后全平台将统一使用该币种进行所有交易和显示"。
3. 若需切换币种，点击当前使用的币种卡片，在弹窗中选择目标币种并确认：
   - 选择新的币种（如 USD / CNY 等）；
   - 确认后，平台所有交易、费用显示将统一切换为该币种。

#### 参数说明

| 字段名称 | 字段类型 | 示例 | 说明 |
|----------|----------|------|------|
| 当前使用币种 | 文本 | `USD` | 平台当前生效的统一结算币种 |
| 货币符号 | 文本 | `$` | 该币种对应的符号标识，用于价格展示 |

## 其他操作

| 操作名称 | 操作步骤 |
|----------|----------|
| 查看当前币种信息 | 进入「币种设置」页面，即可查看当前生效币种及对应货币符号 |

## 注意事项

* 币种切换将影响全平台所有交易和费用显示，请确认后再操作。
* 切换币种不会影响历史交易记录，但显示值将按新币种重新换算。
