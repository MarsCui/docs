# 月度结算单

::: info 文档信息
版本：v1.0
更新日期：2026-07-23
:::

## 功能概述

月度结算单用于按账期查看 Provider 收益结算记录，重点核对结算状态、应结算金额、实际到账金额、抹零、到账时间和详情入口。Provider 可在该页面筛选目标账期，查看结算单详情，并结合收益总览和收益账户出账核对到账结果。

| 项目 | 内容 |
| --- | --- |
| 适用角色 | Provider 账号、Provider 财务查看人员、收益结算处理人员 |
| 导航路径 | 账务 > Provider 收益 > 月度结算单 |
| 页面路由 | `/billing/provider/settlements` |
| 管理对象 | 月度结算单、账期、结算状态、应结算、实际到账、抹零和到账时间 |
| 典型途径 | 查看结算单详情、核对结算金额和到账状态、与收益总览和收益账户出账进行对账 |

#### 新手理解

月度结算单可以理解为 Provider 收益的月度账单。每一行通常对应一个账期，页面展示该账期从应结算到实际到账的处理结果。核对时不要只看单个金额字段，应同时查看状态、抹零、到账时间和详情信息。

#### 术语速查

| 术语 | 含义 | 处理建议 |
| --- | --- | --- |
| 账期 | 月度结算对应的月份或统计周期。 | 与收益总览、收益账户出账保持同一账期口径。 |
| 应结算 | 按结算规则计算出的应结算金额。 | 与收益总览和结算详情核对。 |
| 实际到账 | 实际进入目标账户的金额。 | 结合到账时间、抹零和出账记录核对。 |
| 抹零 | 结算过程中因规则产生的舍入或调整金额。 | 比对应结算和实际到账时必须纳入计算。 |
| 到账时间 | 结算金额完成到账的时间。 | 状态处理中时不要提前判断到账异常。 |
| 转入 | 将满足条件的结算金额转入可用余额的入口。 | 属于高风险动作，只在确认账期、金额和权限后处理。 |

## 前提条件

1. 当前账号具备查看 `Provider 收益 > 月度结算单` 的权限。
2. 已确认需要查看的 Provider 收益账期。
3. 需要核对金额时，已准备好收益总览和收益账户出账等关联页面作为对照。

::: warning 高风险操作边界
`转入`、结算确认或导出真实结算数据会影响收益账户或泄露敏感账务信息。学习或截图时只查看列表字段和详情入口，不执行转入、结算确认或导出。
:::

## 主要操作

### 查看结算单详情

1. 进入 `账务 > Provider 收益 > 月度结算单`。
2. 在列表中查看月度结算单记录。
3. 按需使用 `全部状态` 或 `账期` 筛选目标结算单。
4. 在目标行核对账期、状态、应结算、实际到账、抹零和到账时间。
5. 点击行内 `详情`，查看结算单详细信息。
6. 在详情页或详情区域核对结算构成、到账信息、处理状态和异常提示。
7. 如仅学习或截图，只查看列表字段和详情入口，不执行转入、结算确认或导出真实结算数据。

## 踩坑提示

- 月度结算单涉及收益、到账金额、账期和结算状态，截图、导出、工单和评论必须脱敏。
- `转入`、结算确认或导出真实结算数据属于高风险动作。
- 不要只依据单个金额字段判断异常，应结合应结算、实际到账、抹零、到账时间和结算详情核对。
- 当前账期仍在处理时，结算金额、到账状态和到账时间可能继续变化。
- 不记录真实客户名、账号、金额、流水号、结算单号、到账账户、Token 或 Key。

## 常见问题

#### 查询不到目标账期

**问题现象：**

选择账期后列表为空，或没有看到预期结算单。

**可能原因：**

筛选状态与账期不匹配，或该账期尚未生成可见结算记录。

**处理方式：**

点击 `重置` 后重新选择账期；如仍为空，返回收益总览确认该账期是否已有收益和结算提示。

#### 应结算与实际到账不一致

**问题现象：**

某账期的应结算金额与实际到账金额不同。

**可能原因：**

结算过程中存在抹零、状态处理中，或到账信息尚未完成更新。

**处理方式：**

查看该行 `抹零`、`到账时间` 和 `详情` 信息，必要时结合收益账户出账继续核对。

#### 转入按钮无法继续

**问题现象：**

点击 `转入` 后无法继续，或页面提示当前结算单不能转入。

**可能原因：**

当前账期状态不满足转入条件，或账号缺少处理权限。

**处理方式：**

核对结算单状态和账号权限；未满足条件时不要绕过页面提示继续处理。

## 后续操作

1. 需要查看收益来源时，进入 [收益总览](../revenue/)。
2. 需要核对到账流水时，查看收益账户出账。
3. 需要处理转入或结算确认时，先完成账期、金额、到账账户和权限复核。

## 注意事项

- 月度结算单涉及收益和到账信息，不要在外部渠道传播未脱敏截图。
- `转入`、结算确认和导出真实结算数据可能改变资金状态或扩大敏感数据暴露范围。
- 学习或截图时只查看列表字段和详情入口，不执行转入或结算确认。
- 结算状态与到账时间可能存在处理延迟，核对时应结合详情和收益账户出账。
