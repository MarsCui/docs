# 用户手册动图素材规划

本文仅作为内部素材规划清单，不作为发布手册正文操作步骤。

| 页面 | 建议素材 | 时长 | 优先级 |
| --- | --- | --- | --- |
| On-Prem 集群注册 | 从进入页面到提交集群 | 30s | P1 |
| On-Cloud 快速部署 | 选择模型、推荐规格、提交部署 | 30s | P1 |
| Model Services 文本体验 | 选择模型、输入 Prompt、调整参数 | 30s | P1 |

## 制作要求

1. 使用中文界面录制。
2. 文件名使用小写英文和连字符。
3. 遮挡账号、Token、AK/SK、API Key、Endpoint、租户名称、客户数据和内部资源编号。
4. 每段素材控制在一个核心任务内，不混入无关页面。
