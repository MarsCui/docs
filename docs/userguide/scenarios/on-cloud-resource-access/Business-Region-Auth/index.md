# Authorize Business Types to Regions

Grant each business type access to the cloud regions it may use.

## Procedure

1. Open `Authorization Management > Business-Region Authorization`.
2. Select the business type and cloud platform.
3. Grant the approved regions and save.
4. Reopen the record and verify the effective region set.

See [Business-Region Authorization](../../../../usermanual/ai-infra-on-cloud/operator/auth-management/business-region-auth/).

## Completion Checklist

- [ ] The business type has only the intended regions.
- [ ] Saved authorization persists after refresh.
- [ ] User deployment options follow the granted region set.
