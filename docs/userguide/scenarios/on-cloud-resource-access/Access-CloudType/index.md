# Connect a Cloud Platform

Register the cloud-platform type before connecting accounts and regions.

## Procedure

1. Open `Access Management > Cloud Platforms`.
2. Add the provider identifier and localized display names.
3. For a private cloud, configure the platform access URL required by the deployment.
4. Enable the platform and verify that it appears when adding accounts.

See [Cloud Platform Access](../../../../usermanual/ai-infra-on-cloud/operator/access-management/access-cloudtype/).

## Completion Checklist

- [ ] Platform identifier and display names are correct.
- [ ] Private-cloud access URL is reachable when applicable.
- [ ] The platform is available in account creation.
