# 客户总览

::: info 文档信息
版本：v1.0
更新日期：2026-07-10
:::

## 功能概述

客户总览用于按客户维度查看 Provider 收益情况，包括客户总数、账期新增、Top 客户、Top 5 占比、客户标签、本账期收益、占比、客户加入时间和账期最近收益。服务商可通过账期、关键词和标签筛选客户，并进入客户详情核对明细。

| 项目 | 内容 |
| --- | --- |
| 适用角色 | 服务商账号、客户运营人员、收益分析人员 |
| 菜单路径 | Provider 收益 > 客户总览 |
| 页面用途 | 查看客户维度收益、筛选客户、管理客户标签、进入客户详情 |
| 关键入口 | 搜索、重置、管理标签、刷新统计、详情 |

### 新手理解

客户总览像服务商客户贡献看板。先选择账期，再通过关键词或标签缩小客户范围，最后查看每个客户在该账期的收益、占比和最近收益情况。需要进一步核对客户明细时，进入行内详情。

### 术语速查

| 术语 | 含义 | 处理建议 |
| --- | --- | --- |
| 客户收益 | 某客户在指定账期贡献的收益金额。 | 与收益总览和结算单交叉核对。 |
| 客户标签 | 用于标记 VIP、高潜、试用、待跟进等客户分类。 | 筛选前确认标签含义。 |
| Top 客户 | 当前账期收益贡献较高的客户。 | 不要只凭排名判断客户价值。 |
| 本账期收益 | 客户在所选账期内产生的收益。 | 比较前先统一账期。 |
| 占比 | 客户收益在当前筛选范围内的比例。 | 标签筛选会改变占比口径。 |
| 客户详情 | 查看客户维度收益明细的入口。 | 涉及敏感信息时确认权限。 |

## 前提条件

1. 当前账号具备查看 Provider 客户收益的权限。
2. 已进入 `Provider 收益 > 客户总览`。
3. 查看客户详情或管理标签前，已确认操作权限和客户范围。

## 主要操作

### 查询客户收益

1. 进入 `Provider 收益 > 客户总览`。
2. 在账期选择器中选择目标账期。
3. 如需按客户定位，在 `关键词` 中输入客户相关关键词。
4. 点击 `搜索` 查看结果。
5. 如需恢复默认列表，点击 `重置`。

### 使用标签筛选客户

1. 在客户总览页面找到 `按标签筛选` 区域。
2. 点击 `VIP`、`高潜`、`试用`、`待跟进`、`流失风险` 等标签。
3. 查看列表有没有只展示符合标签条件的客户。
4. 如需取消筛选，重新选择标签或点击 `重置`。

### 查看客户详情

1. 在客户列表中定位目标客户。
2. 核对该行的本账期收益、占比、客户加入时间和账期最近收益。
3. 点击行内 `详情`。
4. 在详情页或详情区域查看客户维度的收益明细。
5. 如涉及敏感客户信息，只在具备权限的范围内查看和使用。

### 管理客户标签

1. 点击页面右侧 `管理标签`。
2. 查看标签管理入口展示的标签项和操作范围。
3. 修改或应用标签前，确认该操作是否会影响客户分类或运营跟进。
4. 不需要变更时，关闭弹窗或返回客户总览。

## 踩坑提示

- 标签筛选会改变列表范围和收益占比，核对前要确认当前是否启用了标签。
- 客户收益占比按当前账期和筛选范围计算，不代表全平台口径。
- 管理标签可能影响客户分类和运营跟进，不要在只排查收益时误保存标签变更。
- 客户详情可能包含组织、管理员和收益数据，查看和外发前都要确认权限和脱敏要求。

## 常见问题

### 客户列表为空

**问题现象：**

进入客户总览后没有看到客户记录。

**可能原因：**

账期、关键词或标签筛选条件过窄，或当前账号没有对应客户范围。

**处理方式：**

点击 `重置` 后重新选择账期；如仍为空，返回收益总览确认该账期是否已有客户收益。

### 客户收益占比看起来异常

**问题现象：**

某个客户的本账期收益或占比与预期不一致。

**可能原因：**

账期选择不同、客户标签筛选改变了统计范围，或最近收益尚未完全反映到列表。

**处理方式：**

确认账期和筛选条件，再进入 `详情` 查看客户收益明细，并结合月度结算单核对。

### 管理标签前需要注意什么

**问题现象：**

点击 `管理标签` 后出现标签管理入口。

**可能原因：**

标签会影响客户筛选、运营分类和后续跟进。

**处理方式：**

修改前确认标签含义和客户范围；只查看时不要保存标签变更。

- 客户收益核对时，应同时查看账期、筛选条件和结算单状态。
