# Model Templates

Model templates combine authors, providers, meta-models, model-source identifiers, modalities, protocols, and defaults into a reusable publishing starting point.

## Procedure

1. Open `Settings > Model Templates` and select the author and provider information.
2. Select the meta-model and model source and configure source model ID.
3. Confirm modalities, protocols, capabilities, token limits, and defaults.
4. Preview the template and publish it to the intended provider scope.

See [Model Templates in the User Manual](../../../../usermanual/model-services/operator/settings/model-templates/).

## Completion Checklist

- [ ] Referenced author, meta-model, and source are available.
- [ ] Previewed capabilities and protocol defaults are correct.
- [ ] Intended providers can select the template.
