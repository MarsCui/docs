# 接入账号

本任务由平台用户添加自己的云账号，供快速部署在已授权地域中创建资源。

## 场景目标

用户云账号校验成功，并能在快速部署的授权范围内被选择。

## 适用角色

- 平台用户

## 开始前准备

- 平台运营方已为租户授权目标云平台和地域。
- 已准备专用的最小权限云凭证。

## 操作步骤

### 添加云账号

1. 进入平台首页，点击左侧导航栏的 **"接入账号"** 菜单，进入接入账号管理页面。
2. 点击页面右上角的 **"添加云账号"** 按钮，弹出「新增账号」窗口。
![接入管理](./images/access-accounts-list.png)
3. 配置账号信息：
   - 填写 **"账号名称"**（如 `aliyun-wh-dev`）；
   - 从下拉列表中选择当前版本已支持并已授权的云平台（如阿里云或 AWS）；华为云接入当前暂不支持；
   - 输入目标云平台的 **"Access Key ID"**（如 `LTAI5tM8xHnXoLuBW...`）；
   - 输入目标云平台的 **"Access Key Secret"**（如 `flsBCIDPLksdaNh05J...`）。
4. 确认所有信息配置无误后，点击 **"确定"** 按钮完成添加；如需放弃，点击 **"取消"**。
![新增账号](./images/add-account.png)
#### 参数说明

| 字段名称 | 字段类型 | 示例 | 说明 |
|----------|----------|------|------|
| 账号名称 | 文本 | `aliyun-wh-dev` | 必填，自定义账号标识 |
| 选择云平台 | 下拉选择 | `阿里云` | 必填，选择账号所属的云平台 |
| Access Key ID | 文本 | `LTAI5tM8xHnXoLuBW...` | 必填，云平台的访问密钥 ID |
| Access Key Secret | 密码 | `flsBCIDPLksdaNh05J...` | 必填，云平台的访问密钥 Secret |

## 完成检查

> **用途：** 以下检查是当前功能任务的退出条件，用于判断操作结果是否可观察、可复核，以及是否可以继续当前场景的下一步。它不是操作步骤的重复；任一项不满足时，请按下方“常见失败分支”继续排查。

| 检查项 | 通过标准 |
| --- | --- |
| 1 | 账号校验成功。 |
| 2 | 账号仅在预期用户或租户范围可见。 |
| 3 | 快速部署可在授权地域使用该账号。 |

## 常见失败分支

| 现象 | 优先检查 |
| --- | --- |
| 账号校验失败 | 凭证有效性、云权限、网络访问和云平台选择 |
| 快速部署选不到账号 | 租户授权、地域范围和账号校验状态 |
