# Cloud Model Assets

Assemble a meta-model, cloud deployment point, cloud model, compute plan, and output definition into a deployable asset.

## Procedure

1. Open `Deployment Assets > Models` and select the meta-model and version.
2. Add the cloud deployment point and assign the cloud model.
3. Select compatible compute plans and configure outputs and API examples.
4. Review the complete configuration and publish the asset.

See [Cloud Model Assets](../../../../usermanual/ai-infra-on-cloud/operator/deploy-assets/models/).

## Completion Checklist

- [ ] Meta-model, deployment point, cloud model, and plan are compatible.
- [ ] Output protocol and example use placeholders instead of real credentials.
- [ ] The asset appears in regular-user quick deployment.
