# 接入资源池

## 前言

| 项目 | 内容 |
|------|------|
| 适用角色 | 运营方 |
| 导航路径 | 接入管理 > 接入资源池 |
| 功能定位 | 维护云平台下的资源池地域（Region）配置，启用/禁用具体地域并管理其显示名称，为后续部署模型、配置算力等流程提供基础数据支撑 |

## 页面结构

### 云平台列表

页面左侧展示云平台列表，可通过 **"输入名称搜索"** 快速筛选云厂商（包含 阿里云 26 个 / 谷歌云 41 个 / 华为云 24 个 / aws 亚马逊 34 个 / AGIOne-powerone 1 个 等）。

### 地域卡片网格

右侧展示选中云平台的所有地域卡片网格（每行 3 列），每张卡片显示地域名称与启用/禁用开关。

### 操作按钮区

- 页面顶部提供 **"导出"** / **"导入"** 按钮，批量管理资源池配置。
- 页面右上角提供 **"只看已启用"** 复选框，用于筛选已启用的地域。

## 操作步骤

### 启用 / 禁用资源池地域

1. 进入平台首页，点击左侧导航栏的 **"接入管理 > 接入资源池"** 菜单，进入资源池管理页面。
2. 在左侧云平台列表上方输入关键字搜索云厂商（包含 阿里云 26 个 / 谷歌云 41 个 / 华为云 24 个 / aws 亚马逊 34 个 / AGIOne-powerone 1 个 等），点击目标云平台（如 阿里云）。
3. 右侧展示选中云平台的所有地域卡片网格（每行 3 列），地域包括 华北1(青岛)、华北2(北京)、华北3(张家口) 等。
4. 找到需要启用 / 禁用的地域，点击地域卡片右侧的开关按钮：
   - 开关为开启状态时，表示该地域已启用；
   - 点击开关即可切换地域的启用 / 禁用状态。

![接入资源池](./images/resource-pools-list.png)

### 编辑地域显示名称

1. 进入平台首页，点击左侧导航栏的 **"接入资源池"** 菜单，进入资源池管理页面。
2. 在左侧选择目标云厂商（如 阿里云），找到需要编辑的地域。
3. 点击地域卡片上的编辑图标，弹出「编辑名称」窗口。
4. **"显示名称"**（标注"多语言"）：用于设置地域在列表、详情和选择控件中的显示名称。点击 **"English"** / **"中文简体"** 标签切换语言 Tab，**"将在 English 语言环境下显示"** / **"将在中文简体语言环境下显示"** 的提示，分别在对应 Tab 下填写英文与中文简体环境下的名称。
5. 确认配置无误后，点击 **"确定"** 按钮完成修改；如需放弃，点击 **"取消"**。

![编辑名称](./images/edit-name.png)

#### 参数说明

| 字段名称 | 字段类型 | 示例 | 说明 |
|----------|----------|------|------|
| 云厂商 | 选择框 | `阿里云` / `亚马逊` | 必填，选择要管理的云厂商资源池 |
| 地域开关 | 开关 | `开启` / `关闭` | 用于启用或禁用对应地域的资源池 |
| 显示名称 | 多语言文本 | English: `China (Beijing)` / 中文简体: `北京` | 必填，分别在"English"和"中文简体"Tab 下配置显示名称 |

## 其他操作

| 操作名称 | 操作步骤 |
|----------|----------|
| 编辑地域显示名称 | 点击目标地域卡片上的编辑图标 → 修改 English / 中文简体 显示名称 → 点击 **"确定"** |
| 只看已启用 | 勾选页面右上角的 **"只看已启用"** 复选框 → 仅显示已启用的地域 |
| 导出 / 导入配置 | 点击页面右上角的 **"导出"** / **"导入"** 按钮 → 批量管理资源池配置 |

## 注意事项

- 地域的启用/禁用状态影响下游模型部署、算力配置等流程，禁用地域将无法在后续流程中使用。
- 多语言字段需同时维护英文与中文两个版本，切换语言 Tab 可维护另一个语言版本。
- 公有云场景下地域以云平台官方命名为准（如"华东2（上海）"）；私有云场景下地域命名由平台自定义。
