# 语音生成

## 前言

| 项目 | 内容 |
|------|------|
| 适用角色 | 普通用户 |
| 导航路径 | 体验中心 > 语音生成 |
| 功能定位 | 通过文本内容生成语音，体验模型的文本转语音能力 |

## 页面结构

### 搜索区域

主页面无搜索区域；「选择模型」弹窗左侧提供模型名称 / 模型标识搜索框。

### 操作按钮区

* 顶部模型下拉框用于查看当前语音模型并打开「选择模型」弹窗
* 右上角提供「多个模型对比」和「返回旧版」按钮
* 输入框左下角提供 Voice ID 选择和参数配置按钮
* 输入框右下角提供调用密钥选择和发送按钮

### 数据列表说明

页面中央展示语音生成提示区、文本输入框、参数配置入口和生成结果区域。

## 操作步骤

### 模型生成音频

1. 进入平台首页，点击 **"模型及AI服务 > 体验中心 > 语音生成"** 菜单，进入语音生成页面。
2. 点击顶部模型下拉框，打开「选择模型」弹窗：
   - 可在左侧搜索框输入模型名称或模型标识进行筛选；
   - 在左侧模型列表中选择目标语音模型；
   - 在右侧供应方区域选择可用供应方实例；
   - 点击「确定」完成模型切换。

![选择模型](./images/select-model.png)

3. 在输入框左下角选择 **Voice ID**，指定本次语音生成使用的音色。
4. 点击参数配置按钮，设置生成参数：
   - 选择 **协议**（如 openai/audio）；
   - 填写必填的 **Text**；
   - 调整 **Speed**、**Volume**、**Pitch**；
   - 按需设置 **Audio Format**。

![语音生成参数配置](./images/parameters.png)

5. 如需指定调用凭证，在输入框右下角选择对应的 **Personal Key**。
6. 在输入框中输入或确认待合成文本，点击发送按钮生成音频。
7. 生成完成后，在页面结果区域查看并播放生成的音频。

#### 参数说明（参数配置面板）

| 字段名称 | 字段类型 | 示例 | 说明 |
|----------|----------|------|------|
| Voice ID | 下拉选择 | `male-qn-qingse` | 语音生成使用的音色标识 |
| 协议 | 下拉选择 | `openai/audio` | 模型调用的 API 协议 |
| Text | 文本输入 | `please input` | 必填，待合成的文本内容 |
| Speed | 数值滑块 / 输入框 | `1.0` | 控制生成音频的语速 |
| Volume | 数值滑块 / 输入框 | `1.0` | 控制生成音频的音量 |
| Pitch | 数值滑块 / 输入框 | `0` | 控制生成音频的音调 |
| Audio Format | 下拉选择 | `按页面选项选择` | 输出音频的文件格式 |
| Personal Key | 下拉选择 | `Personal Key 20260616...` | 当前请求使用的调用凭证 |

#### 参数说明（模型选择弹窗）

| 字段名称 | 字段类型 | 示例 | 说明 |
|----------|----------|------|------|
| 模型名称 / 标识 | 文本 | `Mock MiniMax Speech T2A / mock-models/speech-2.8-hd` | 模型的名称与唯一标识 |
| 发布日期 | 日期 | `2026-06-02` | 模型的发布时间 |
| 上下文长度 | 文本 | `-` | 语音模型的上下文长度展示值 |
| 输入 Credit | 数值 | `3500 Credit` | 模型输入侧的计费展示 |
| 供应方 | 文本 | `Model Mocker:Mock MiniMax Speech T2A / AGIOneSystem` | 模型的供应方及服务实例 |
| 计费单位 | 文本 | `Per Character` | 当前供应方实例的计费方式 |
| 字符价格 | 数值 | `3,500 Credits/M chars` | 每百万字符的调用费用 |
| 延迟 / 吞吐量 / 成功率 | 数值 | `- / 0 t/s / -` | 供应方实例的运行指标 |
| 周调用量 / 月字符量 | 数值 | `0 / 0 chars` | 该供应方实例的近期使用情况 |
| 状态标签 | 标签 | `推荐 / 已上架` | 模型或供应方实例的推荐和上架状态 |
| 能力标签 | 标签 | `深度思考 / 工具调用` | 模型能力标签；截图中均为不支持状态 |

### 模型对比

1. 点击 **"多个模型对比"** 按钮，进入多模型对比页面。
2. 选择需要对比的语音模型或供应方实例。
3. 分别配置各模型的 Voice ID、协议和生成参数。
4. 输入相同的待合成文本。
5. 点击发送按钮，生成多组音频结果。
6. 对比不同模型或供应方生成音频的音色、语速、稳定性和效果。

## 其他操作

| 操作名称 | 操作步骤 |
|----------|----------|
| 切换模型 | 点击顶部模型下拉框 → 在弹窗中选择不同模型或供应方 → 点击「确定」 |
| 搜索模型 | 在「选择模型」弹窗左侧搜索框输入模型名称或模型标识，快速定位目标模型 |
| 切换音色 | 点击输入框左下角 Voice ID 下拉框，选择本次生成使用的音色 |
| 调整参数 | 点击输入框左下角参数配置按钮 → 修改协议、Text、Speed、Volume、Pitch、Audio Format 等参数 |
| 选择调用密钥 | 点击输入框右下角密钥下拉框，选择本次请求使用的 Personal Key |
| 多个模型对比 | 点击「多个模型对比」按钮，进入多模型并行语音生成体验页面 |
| 文本输入 | 在底部输入框输入文本，按发送按钮或 Enter 发送；使用 Shift+Enter 换行 |
| 返回旧版 | 点击「返回旧版」按钮，切换到旧版语音生成页面 |

## 注意事项

* Text 为必填项，未填写时页面会提示需要输入文本。
* Voice ID 决定生成音频的音色，生成前请确认已选择合适音色。
* Speed、Volume、Pitch 会影响语音的播放速度、音量和音调。
* Audio Format 的可选项以页面实际下拉选项为准。
* 模型价格、状态、计费单位和性能指标以「选择模型」弹窗中展示的信息为准。
* 可点击「多个模型对比」按钮进入多模型并行语音生成体验页面。
