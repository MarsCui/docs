# Connect a Cloud Account

Connect an operator-managed cloud account used to discover and deploy regional resources.

## Procedure

1. Open `Access Management > Cloud Accounts` and add an account.
2. Select the cloud platform and enter the account identity and secret through the protected form.
3. Validate the account and review the permissions required by AGIOne.
4. Confirm that available regions can be discovered.

See [Cloud Accounts](../../../../usermanual/ai-infra-on-cloud/operator/access-management/cloud-accounts/).

## Completion Checklist

- [ ] Account validation succeeds.
- [ ] Only required cloud permissions are granted.
- [ ] Regions and resources are discovered as expected.
