# 币种设置

## 功能概述

`币种设置` 用于维护或查看币种、汇率、展示符号、结算口径和启用状态，支撑模型发布、体验、调用、统计和运营治理。

| 项目 | 内容 |
| --- | --- |
| 适用角色 | 运营方 |
| 导航路径 | 系统设置 > 币种设置 |
| 页面路由 | /operator/settings/currency-settings |
| 管理对象 | 币种、汇率、展示符号、结算口径和启用状态 |
| 典型用途 | 维护模型市场计费展示与收益换算口径 |

### 新手理解

货币设置像平台里的计价尺，决定用量如何折算成点数、余额或收益。口径不清会导致用量、收益和客户账单对不上。

### 术语速查

| 术语 | 说明 |
| --- | --- |
| 币种 | 价格展示或结算使用的货币单位。 |
| 汇率 | 不同币种之间的换算比例。 |
| 结算口径 | 收益、用量或费用统计时采用的计算规则。 |
| 启用状态 | 控制币种是否在页面中可用。 |

## 前提条件

1. 当前账号具备货币或点数设置权限。
2. 计量单位、换算比例、展示精度和适用业务范围已确认。
3. 收益、用量和账单导出口径已与财务或运营口径对齐。
## 页面说明

页面用于维护计费币种、点数名称、换算比例、展示精度和启用状态。运营方应与计费、收益和用量统计口径保持一致，并避免在文档中暴露真实商业价格策略。

## 主要操作

### 操作步骤

1. 进入 `系统设置 > 货币设置`。
2. 查看当前币种、点数名称和换算比例。
3. 新增或编辑时填写展示名称、单位、精度和启用状态。
4. 保存前与用量统计和收益页面口径核对。
5. 保存后在模型收益和模型用量页面验证展示结果。

### 参数说明

| 字段名称 | 是否必填 | 字段类型 | 示例 | 说明 |
| --- | --- | --- | --- | --- |
| 展示名称 | 是 | 文本 | `Credits` | 用户看到的计量单位名称。 |
| 换算比例 | 是 | 数字 | `1:1000` | 内部计量值到展示单位的换算规则。 |
| 精度 | 是 | 数字 | `2` | 页面展示的小数位。 |
| 适用范围 | 否 | 多选 | `模型调用` | 该货币配置影响的业务范围。 |
| 状态 | 是 | 枚举 | `启用` | 控制是否生效。 |

### 踩坑提示

- 不要把真实商业折扣、客户协议价或内部结算价写入截图。
- 换算比例变更会影响收益和用量展示，发布前要做口径说明。
- 同一页面不要混用多个单位名称。

### 结果校验

1. 用量、收益页面展示的单位与货币设置一致。
2. 明细导出中的原始计量值和展示值可对账。
3. 配置变更后新统计周期按新口径展示。
## 常见问题

### 收益金额与用量不一致

**问题现象：**

模型收益页和模型用量页折算结果对不上。

**可能原因：**

- 换算比例或精度配置不一致。
- 统计时间范围不同。
- 存在延迟结算或退款调整。

**处理方式：**

1. 核对货币设置和统计时间范围。
2. 查看导出明细中的原始计量值。
3. 等待结算任务完成后复核。

### 页面单位展示异常

**问题现象：**

页面显示旧单位或小数位不符合预期。

**可能原因：**

- 配置未启用。
- 页面缓存未刷新。
- 部分模块仍使用旧配置。

**处理方式：**

1. 确认新配置启用状态。
2. 刷新页面或等待缓存更新。
3. 检查相关模块是否支持该单位。

## 后续操作

1. 查看模型收益页面。
2. 查看模型用量页面。
3. 同步更新对外计费说明。

## 注意事项

- 不要在截图中暴露真实客户协议价、折扣或结算金额。
- 换算比例变更需要同步说明统计周期和生效时间。
- 对外材料优先使用脱敏样例金额。
