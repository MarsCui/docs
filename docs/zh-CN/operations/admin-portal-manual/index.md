# 管理门户手册

本章节说明管理门户中的核心管理流程。当前版本聚焦用户全生命周期管理、角色授权、账号状态控制和审计校验。

## 适用对象

| 角色 | 典型任务 |
|------|----------|
| 平台管理员 | 创建用户、维护账号状态、分配平台角色。 |
| 租户管理员 | 在授权租户或组织范围内管理用户。 |
| 安全或审计负责人 | 确认用户变更可追溯，权限边界符合预期。 |

## 文档清单

| 文档 | 用途 |
|------|------|
| [用户管理](./user) | 说明用户创建、角色授权、状态控制、重置和审计检查等管理操作 |

## 操作原则

- 按任务需要分配最小角色和最小资源范围。
- 创建或更新用户前，先确认目标租户、组织或项目。
- 日常操作避免使用共享账号。
- 禁用用户、重置访问等高风险变更需要可通过审计日志追溯。
- 权限调整后，应重新检查用户实际登录结果和可见菜单。
