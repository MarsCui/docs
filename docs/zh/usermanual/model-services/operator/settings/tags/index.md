# 标签

## 前言

| 项目 | 内容 |
|------|------|
| 适用角色 | Operator |
| 导航路径 | 设置 > 标签 |
| 功能定位 | 定义和管理应用、模型、工具、知识库的分类标签体系 |

## 页面结构

### 搜索区域

页面顶部提供筛选功能，支持按标签类型（应用 / 模型 / 工具 / 知识库）和状态（全部 / 可用 / 不可用）进行筛选，或直接输入名称搜索。

### 操作按钮区

* 页面右上角提供 **"新增标签"** 按钮，用于创建新标签
* 每个标签提供 **"编辑"**、**"添加"**（子标签）、**"启用" / "禁用"**、**"删除"** 操作按钮

### 数据列表说明

页面以树形结构展示标签列表，支持一级标签和子标签的层级展示。

## 操作步骤

### 添加标签

1. 进入平台首页，点击左侧导航栏的 **"设置 > 标签"** 菜单，进入标签管理页面。
2. 点击页面右上角的 **"新增标签"** 按钮，弹出「新增标签」窗口：
3. 填写 **编码**（如 `text_generation`）；
4. 配置 **多语言名称**（分别填写英文与中文简体环境下的名称）；
5. 配置 **多语言备注**；
6. 点击 **"确定"** 完成添加。

#### 参数说明

| 字段名称 | 字段类型 | 示例 | 说明 |
|----------|----------|------|------|
| 编码 | 文本 | `text_generation` | 必填，标签的唯一标识 |
| 名称（多语言） | 文本 | `Text Generation / 文本生成` | 必填，需分别配置英文与中文简体环境下的显示名称 |
| 备注（多语言） | 文本 | `描述文本` | 选填，标签的补充说明信息，支持多语言配置 |

## 其他操作

| 操作名称 | 操作步骤 |
|----------|----------|
| 筛选与搜索 | 通过顶部标签类型（应用 / 模型 / 工具 / 知识库）和状态（全部 / 可用 / 不可用）筛选，或直接输入名称搜索 |
| 编辑标签 | 点击目标标签的 **"编辑"** 按钮 → 修改编码、多语言名称、备注等信息 → 点击 **"确定"** |
| 添加子标签 | 点击目标一级分类的 **"添加"** 按钮 → 填写子标签信息 → 点击 **"确定"** |
| 启用 / 禁用 | 点击目标标签的 **"启用"** / **"禁用"** 按钮 → 确认状态变更 |
| 删除标签 | 点击目标标签的 **"删除"** 按钮 → 删除操作不可逆，请谨慎操作 |

## 注意事项

* 标签编码一旦保存不可修改，请谨慎填写。
* **删除操作不可逆**，请谨慎操作，删除后数据将无法恢复。
* 禁用标签后，相关联的应用或模型将不再显示该标签。
