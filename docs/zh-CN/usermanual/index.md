## 手册总览

用户手册面向平台运营、模型提供方、普通用户、账务运营和系统管理员，覆盖模型发布与调用、云上资源调度、本地异构算力纳管、账务处理和平台设置五类日常操作。

当前手册按左侧导航分为五个模块：

1. 模型及AI服务：发布、审核、体验和调用模型。
2. 多云调度平台：接入云账号、授权资源池并部署云上模型服务。
3. 异构卡纳管：纳管本地资源池、模板、配额、监控和用户侧资源。
4. 账务：查看用户账务、客户账务、运营财务和 License。
5. 设置：管理账号、成员、角色、组织、系统配置和操作日志。

## 推荐学习顺序

1. 先读本页，确认自己属于哪类角色、要进入哪个模块。
2. 再读对应模块的入门页，理解核心对象、前提条件和常见限制。
3. 如果任务跨模块，优先读端到端流程页，再进入具体功能页执行操作。
4. 已经知道目标页面时，直接进入功能页查看页面说明、操作步骤、参数说明和结果校验。
5. 遇到异常时，优先查对应功能页的常见问题、注意事项、日志、事件、监控或账务流水。

## 使用前提

1. 已确认当前账号角色、所属租户和可见组织范围。
2. 已确认目标模块、资源池、云账号、模型、账期或设置对象对当前账号可见。
3. 涉及部署、授权、充值、结算、调账、删除、发布或 License 生效的操作，应先确认影响范围和回退方式。
4. 截图、工单和沟通材料不得包含密码、Token、AK/SK、私钥、完整凭据、客户敏感数据或真实内部地址。

## 常见问题

### 不确定应该读哪个模块

**问题现象：**

想完成模型部署、模型调用、资源接入或账务核对，但不确定从哪个模块开始。

**可能原因：**

- 任务同时涉及资源、模型、权限和账务。
- 当前角色和目标操作之间的边界不清楚。
- 页面入口名称和业务目标不是一一对应关系。

**处理方式：**

1. 需要发布、体验或调用模型，先看模型及AI服务。
2. 需要接入云资源或部署云上模型服务，先看多云调度平台。
3. 需要管理本地集群、实例、模板、存储或配额，先看异构卡纳管。
4. 需要核对余额、流水、账单、结算、调账或 License，先看账务。
5. 需要维护账号、项目、Key、成员、角色、组织或安全策略，先看设置。

### 页面入口不可见

**问题现象：**

文档中提到的菜单、按钮或页面在当前账号下不可见。

**可能原因：**

- 当前账号角色或租户权限不包含该功能。
- 目标资源、云账号、模型、账期或组织范围未对当前账号开放。
- 功能依赖的前置资源尚未配置完成。

**处理方式：**

1. 先确认当前账号角色、租户、组织和模块入口。
2. 对照功能页的前提条件检查资源、授权和可见范围。
3. 提交协作信息时只提供脱敏后的页面路径、时间范围、资源编号和缺失入口说明。

### 操作失败但步骤已经完成

**问题现象：**

按文档完成操作后，实例、部署、作业、调用、账务处理或设置变更仍未达到预期。

**可能原因：**

- 权限、额度、容量、规格、镜像、存储、模型来源或上游服务不满足条件。
- 云账号授权、Personal Key、API Key、请求头或业务配置已失效。
- 监控、计量、账务或状态同步存在延迟。

**处理方式：**

1. 回到对应功能页查看结果校验和注意事项。
2. 结合事件、日志、监控、调用记录、账务流水或操作日志交叉定位。
3. 涉及费用、授权、删除、发布和账务变更时，先暂停最终确认动作并核对影响范围。
