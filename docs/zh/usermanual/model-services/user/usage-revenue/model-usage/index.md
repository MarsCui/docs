# 用量明细

## 前言

| 项目 | 内容 |
|------|------|
| 适用角色 | 普通用户 |
| 导航路径 | 用量与收益 > 用量明细 |
| 功能定位 | 消费用户查看用量概览与消耗详情的入口（英文 UI 标题 **Model Usage**），含 **2 个子页签**：**消耗明细概览**（默认）+ **消耗明细详情** |

## 页面结构

### 顶部工具栏

页面顶部固定区域提供筛选与查询工具：
- **"消耗明细概览" / "消耗明细详情"** Tab 切换（默认 消耗明细概览）
- **"数据粒度"** 下拉选择器（默认 **"天"**）
- **"时间范围"** 选择器（默认 `2026-06-01 00:00:00 - 2026-06-23 15:57:00`）

### 消耗明细概览（默认 Tab）

页面分 2 大区块：3 个核心指标卡片 + 4 张图表（2×2 网格布局）。

- **3 个核心指标卡片**（每张含迷你折线图 + 较上期环比）：
  - **已使用模型数量**（如 `2`，↓ 0% 较上期）— 统计周期内被调用的模型总数
  - **总消耗（Credit）**（如 `0.01`，↓ 0% 较上期）— 统计周期内的总消耗 Credit 数
  - **Token 总消耗**（如 `122`，↓ 0% 较上期）— 统计周期内的总消耗 Token 数
- **4 张图表**（2×2 网格）：
  - **模型消耗趋势**（折线图，多线对比）— 按模型类型（语音模型 / 对话模型 / 图片模型 / 视频模型 / 嵌入模型 / 重排模型）统计的消耗时间序列
  - **模型使用分布**（环形图）— 不同模型类型对总消耗的贡献占比（如 语音模型 0.01 100%）
  - **模型调用次数趋势**（折线图，多线对比）— 按模型实例（如 `--` / `audio-dsy-1.1`）统计的调用量时间序列
  - **模型调用次数分布**（饼图）— 不同模型的调用次数占比

### 消耗明细详情 Tab

页面分 2 段：3 个结算数据卡片 + 7 列消耗明细表格。

- **3 个结算数据卡片**（顶部）：
  - **本账期消耗**（如 `0`）— 当前账期的总消耗
  - **已入账**（如 `0`）— 已完成结算的消耗
  - **待结转 ⊕**（如 `0`）— 当前账期暂未结算的消耗
- **筛选工具栏**（账期下方）：
  - **"账期"** 选择器（默认 `2026-06`，YYYY-MM 格式）
  - 时间范围（默认锁定为账期内的起止时间）
  - **"模型名称"** 输入框
  - **"模型类型"** 下拉
  - **"搜索"** / **"重置"** 按钮
- **7 列消耗明细表格**（按时间倒序，每行展示一次调用的消耗详情）：使用时间 / 模型 / 延迟 / 计量用量 / 免费配额 / 计费模式 / Credit

## 操作步骤

### 查看消耗明细概览（消耗明细概览 Tab，默认）

1. 进入平台首页，点击左侧导航栏的 **"用量与收益 > 用量明细"** 菜单，默认进入 **"消耗明细概览"** Tab。
2. 页面顶部 **"数据粒度"** 选择器（默认 **"天"**）+ **时间范围"** 选择器（如 `2026-06-01 00:00:00 - 2026-06-23 15:57:00`）。
3. 查看 **3 个核心指标卡片**（每张含迷你折线图 + 较上期环比）：
   - **已使用模型数量**（如 `2`，↓ 0% 较上期）— 统计周期内被调用的模型总数
   - **总消耗（Credit）**（如 `0.01`，↓ 0% 较上期）— 统计周期内的总消耗 Credit 数
   - **Token 总消耗**（如 `122`，↓ 0% 较上期）— 统计周期内的总消耗 Token 数
4. 查看 **4 张图表**（2×2 网格布局）：
   - **模型消耗趋势**（折线图，多线对比）：按模型类型（语音模型 / 对话模型 / 图片模型 / 视频模型 / 嵌入模型 / 重排模型）统计的消耗随时间变化；鼠标悬停可显示当日各模型类型的具体消耗值
   - **模型使用分布**（环形图）：不同模型类型对总消耗的贡献占比（如 语音模型 0.01 100%）
   - **模型调用次数趋势**（折线图，多线对比）：按模型实例（如 `--` / `audio-dsy-1.1`）统计的调用量随时间变化
   - **模型调用次数分布**（饼图）：不同模型的调用次数占比

![用量明细](./images/overview.png)

### 查看消耗明细详情（消耗明细详情 Tab）

1. 点击 **"消耗明细详情"** Tab，切换到结算明细视图。
2. 页面顶部 **"账期"** 选择器（默认 `2026-06`，YYYY-MM 格式），切换账期后下方数据按账期刷新。
3. 查看 **3 个结算数据卡片**：
   - **本账期消耗**（如 `0`）— 当前账期的总消耗
   - **已入账**（如 `0`）— 已完成结算的消耗
   - **待结转 ⊕**（如 `0`）— 当前账期暂未结算的消耗
4. 时间范围（默认锁定为账期内的起止时间，如 `2026-06-01 00:00:00 - 2026-06-23 15:58:00`）+ 筛选工具栏（**模型名称** 输入框 + **模型类型** 下拉 + **搜索** / **重置** 按钮）。
5. 查看 **7 列消耗明细表格**（使用时间 / 模型 / 延迟 / 计量用量 / 免费配额 / 计费模式 / Credit），按时间倒序展示每笔调用记录的消耗详情：
   - **使用时间**：调用发生的时间（如 `2026-06-23 09:41:02`）
   - **模型**：模型名 + 模型类型标签（如 `--` + `对话模型` / `audio-dsy-1.1` + `语音模型`）
   - **延迟**：本次调用的耗时（ms，如 `888`）
   - **计量用量**：4 行 Token 信息（**Input Tokens** + **Output Tokens** / **Tokens**）
   - **免费配额**：是否在免费额度内（`--` 表示已用完 / 在额度内）
   - **计费模式**：Token / 阶梯 等
   - **Credit**：本次调用的消耗 Credit（如 `0 credits` / `0.01 credits`）
6. 表格底部为分页控件（如 共 7 条 / 10条/页 / 前往 1 页）。

## 其他操作

| 操作名称 | 操作步骤 |
|----------|----------|
| 切换数据粒度 | 顶部 **"数据粒度"** 选择器切换 `天 / 小时 / 分钟` → 所有图表与卡片按新粒度刷新 |
| 切换时间范围 | 顶部时间范围选择器（默认 `2026-06-01 ~ 2026-06-23`）→ 自定义起止时间后图表与卡片按新范围刷新 |
| 切换账期 | 消耗明细详情 Tab → 顶部 **"账期"** 选择器（默认 `2026-06`，YYYY-MM 格式）→ 切换账期后下方数据按账期刷新 |
| 筛选模型 | 消耗明细详情 Tab → **"模型名称"** 输入框 → 输入关键字后点击 **"搜索"** 过滤该模型的明细 |
| 筛选模型类型 | 消耗明细详情 Tab → **"模型类型"** 下拉选择（如 对话模型 / 图片模型 / 语音模型 等）→ 点击 **"搜索"** 过滤该类型的明细 |
| 重置筛选 | 消耗明细详情 Tab → 点击 **"重置"** 按钮 → 清空模型名称 / 模型类型筛选条件 |
| 查看计费规则详情 | 消耗明细详情 Tab → 在表格中点击 **"View full details"** / **"View all N prices"** 链接 → 查看完整计费规则 |

## 注意事项

- **页面名 vs 菜单名**：菜单显示 **"用量明细"**（英文 Model Usage），页面内 Tab 显示 **"消耗明细概览"** / **"消耗明细详情"**；"消耗明细"是"用量明细"的旧内部名，两者指代同一功能。
- **结算周期**：消耗通常按月结算（账期 = YYYY-MM），具体周期以平台规则为准。
- **数据延迟**：消耗数据通常有 1-2 天的结算延迟，刚发生的调用不会立即体现。
- **本账期 vs 已入账 vs 待结转**：
  - **本账期消耗** = 当前账期的总消耗
  - **已入账** = 已完成结算的消耗
  - **待结转** = 当前账期暂未结算的消耗
- **Token 字段差异**：
  - 对话/图片等模型：分 **Input Tokens** + **Output Tokens**
  - 语音模型：使用单一的 **Tokens**（无 Input/Output 拆分）
- **免费配额标识**：`--` 表示已用完免费额度或在额度内（具体含义需结合计费模式判断）。
