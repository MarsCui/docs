## 操作步骤

按照页面引导，依次完成以下 5 个步骤：

### Step 1：接入云平台

点击「**继续配置**」按钮，注册或添加公有云 / 私有云厂商平台，作为后续所有接入操作的基础。

### Step 2：云平台授权

点击「**继续授权**」按钮，将云平台的使用权限授予指定租户，租户授权后方可使用对应云平台的资源。

### Step 3：接入账号

点击「**继续添加**」按钮，为租户绑定云平台的访问凭证（AK/SK 等），系统将通过账号拉取并管理云端资源。

### Step 4：接入资源池

点击「**继续配置**」按钮，配置可用的云区域与可用区，启用后资源池将可被业务系统正常使用。

### Step 5：资源池授权

点击「**前往授权**」按钮，将资源池的访问权限分配给业务单元，完成后业务系统即可正式调用云端算力资源。

## 注意事项

- 执行接入操作前，请确保已具备目标云平台的管理员权限
- 云平台授权须针对具体租户进行，未授权的租户无法使用对应云资源
- 接入账号时请妥善保管 AK/SK 凭证，避免泄露
- 资源池地域启用后不可随意禁用，以免影响线上业务
