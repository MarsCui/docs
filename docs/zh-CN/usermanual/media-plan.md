## 检查口径

本页用于跟踪中文用户手册的截图和媒体素材状态。检查范围为 `zh-CN/usermanual` 下的 Markdown 图片引用和本地图片文件，不包含英文手册、侧边栏和未引用的外部素材。

媒体检查优先关注：

1. 图片引用是否断链。
2. 关键操作页是否有列表、表单、弹窗或详情截图。
3. 截图是否为中文页面、浅色主题，并裁掉顶部全局菜单。
4. 截图是否避开密码、Token、AK/SK、私钥、完整凭据和客户敏感数据。

## 素材命名规则

1. 图片放在对应页面的 `images` 目录。
2. 文件名使用小写英文、数字和短横线。
3. 列表页建议使用 `*-list.png`。
4. 新增、编辑、详情、弹窗、抽屉截图按动作命名，例如 `create-role.png`、`license-detail.png`。
5. 不使用中文文件名，不在文件名中写账号、客户、密钥或内部地址。

## 后续维护

1. 新增页面时同步创建 `images` 目录，并在关键步骤附近插入截图。
2. 替换截图时优先覆盖原文件名，减少 Markdown 引用变更。
3. 重截后检查图片是否为真实 PNG，确认引用文件存在。
4. 每次大批量改图后运行占位扫描、图片引用检查和构建验证。
