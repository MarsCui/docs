## 功能概述

`概览` 作为普通用户进入异构卡纳管后的总览页面，用于集中查看部署模板、加速卡能力、快捷创建入口、规格配额和资源消费趋势。

| 项目 | 内容 |
| --- | --- |
| 适用角色 | 普通用户 |
| 导航路径 | 概览 |
| 页面路由 | `/powerone/overview` |
| 管理对象 | 部署模板、加速卡资源、快速入口、规格配额、资源用量趋势 |
| 典型用途 | 进入异构卡纳管后快速了解可部署模型、可用加速卡、配额和近期资源使用情况 |

### 新手理解

概览页像异构卡纳管用户控制台的首页：上半部分告诉你当前能部署哪些模型、有哪些加速卡；中部提供在线 IDE 和运行实例的快捷入口；下方帮助你确认配额是否足够、资源有没有消费记录。

### 首次使用流程

1. 进入 `异构卡纳管 > 概览`。
2. 查看 `部署模板（Deployment Templates）`，确认是否有可部署模型。
3. 查看 `加速卡厂商（Accelerator Vendors）`，确认当前资源池支持的加速卡类型。
4. 通过 `在线 IDE（Online IDE）` 或 `运行实例（Runtime Instance）` 快捷入口进入创建流程。
5. 查看规格配额、消费趋势和资源用量，确认是否具备创建作业所需资源条件。

### 术语速查

| 术语 | 说明 |
| --- | --- |
| 规格 | 作业可申请的资源套餐，例如 CPU、内存、GPU 型号和卡数。 |
| 配额 | 租户可使用的资源上限，常见维度包括 GPU、CPU、内存和规格。 |
| 加速卡 | GPU、NPU 或其他 AI 计算设备，在页面中按厂商和型号展示。 |
| 消费趋势 | 按时间统计的资源消耗变化，用于判断近期是否有作业产生费用或额度消耗。 |

## 前提条件

1. 当前账号能进入 `异构卡纳管` 子系统。
2. 运营方已为租户开放可见的地域、可用区、集群、规格和模板。
3. 如需创建在线 IDE、运行实例或模型实例，账号需要具备对应创建权限和可用配额。

## 页面说明

页面上方展示可部署模板和加速卡厂商；中部提供在线 IDE、运行实例快捷入口；下方展示规格配额、消费趋势和资源用量。截图中可看到左侧用户端菜单、部署模板区域、加速卡区域和快速入口。

### 页面区域

| 字段/区域 | 说明 |
| --- | --- |
| 部署模板 | 展示可部署模型和部署入口，未选择模型时部署按钮通常不可用。 |

## 主要操作

### 查看可部署资源

#### 适用场景

当你准备创建模型服务、在线 IDE 或运行实例前，先在概览页确认模板、加速卡和配额是否可用。

#### 操作前确认

1. 已进入异构卡纳管概览页。
2. 确认页面右上角地域或资源范围符合本次使用场景。

#### 操作步骤

1. 在部署模板中查看模型厂商和模型卡片。
2. 在加速卡厂商中查看加速卡厂商、型号、显存和适配状态。
3. 在规格配额中确认目标规格是否仍有可用配额。
4. 如需要创建作业，点击 `创建作业（Create Job）` 进入对应创建页面。

#### 参数说明

| 字段名称 | 是否必填 | 字段类型 | 示例 | 说明 |
| --- | --- | --- | --- | --- |
| 地域 | 否 | 下拉选择 | `武汉` | 概览统计地域。 |
| 资源类型 | 否 | 枚举 | `GPU` | 概览资源类别。 |
| 配额 | 系统生成 | 数字 | `10` | 当前账号可用配额。 |
| 已用量 | 系统生成 | 数字 | `4` | 当前已使用资源量。 |
| 更新时间 | 系统生成 | 日期时间 | `2026-07-07 10:00` | 概览数据更新时间。 |
#### 踩坑提示

- 部署按钮不可用时，通常是尚未选中模型或加速卡。
- 配额充足不等于集群一定有空闲资源，创建失败时还需查看地域、可用区和规格。

#### 结果校验

| 检查项 | 成功表现 | 异常时处理 |
| --- | --- | --- |
| 目标模型或快捷入口可见 | 目标模型或快捷入口可见。 | 未达到时回到对应页面核对权限、筛选条件和配置状态 |
| 目标规格在配额表中可见 | 目标规格在配额表中可见。 | 未达到时回到对应页面核对权限、筛选条件和配置状态 |
| 如有消费数据 | 如有消费数据，趋势和用量区域能正常刷新。 | 未达到时回到对应页面核对权限、筛选条件和配置状态 |

## 结果校验

- 当前页面的目标记录、状态、关联对象或统计结果与本次查看范围一致。
- 完成配置或查看后，返回列表确认页面能正常加载，筛选条件和目标对象仍可定位。
- 涉及资源、配额、监控或存储影响时，继续结合“配置规则与影响”和“注意事项”核对。

## 配置规则与影响

- 概览页只做汇总展示，不替代具体模块的创建、详情和排障页面。
- 模板、规格和加速卡由运营方配置，普通用户无法在概览页直接修改。
- 资源用量和消费趋势可能存在统计延迟，实时排障应进入具体实例或监控页面。

## 常见问题

### 概览页没有可部署模板

**问题现象：**部署模板区域为空或没有可点击模型。

**可能原因：**

- 运营方未发布模板。
- 当前租户没有模板可见权限。
- 筛选条件或地域范围不匹配。

**处理方式：**

1. 进入 `模型部署 > 部署模板` 查看完整列表。
2. 切换或确认当前地域。
3. 联系运营方确认模板是否已发布给当前租户。

### 配额显示充足但创建失败

**问题现象：**概览页显示无限制或仍有配额，但创建作业时失败。

**可能原因：**

- 目标集群空闲资源不足。
- 规格没有关联到可用集群。
- 镜像、存储或地域配置不完整。

**处理方式：**

1. 进入创建页查看具体错误提示。
2. 检查 `资源配额` 和 `资源用量`。
3. 联系运营方确认规格与集群关联关系。

## 后续操作

1. 进入 `模型部署 > 部署模板` 创建模型实例。
2. 进入 `开发资源 > 在线IDE` 创建交互式开发环境。
3. 进入 `配额&用量 > 资源配额` 查看资源上限。

## 注意事项

- 不要仅凭首页汇总判断故障原因，关键实例状态以对应详情页为准。
- 创建前先确认地域、规格和镜像来源，避免提交后因资源不可调度失败。
