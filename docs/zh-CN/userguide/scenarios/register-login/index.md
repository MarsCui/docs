# 场景概览 - 注册与登录

本场景介绍创建账号、密码登录、邮箱验证码登录和找回密码，是进入 AGIOne 各子系统前的统一账号入口。

## 适用角色

- 平台用户（End User）、模型提供方（Provider）、平台运营方（Operator）

## 场景目标

- 完成账号注册或使用已有账号登录。
- 在忘记密码时通过邮箱验证恢复访问。
- 登录后进入账号权限对应的工作台。

## 场景流程

**主线：** 准备访问条件 → 注册或登录 → 验证角色工作台 → 建立账号恢复路径

| 阶段 | 核心结果 |
| --- | --- |
| 1. 准备访问条件 | 获得正确的平台地址、邮箱和所属租户信息 |
| 2. 注册或登录 | 账号完成验证，并通过支持的方式进入平台 |
| 3. 验证工作台 | 当前角色、菜单和可见子系统符合预期 |
| 4. 恢复访问 | 忘记密码或验证码异常时有明确的恢复方法 |

## 开始前准备

- 可访问的企业邮箱。
- 已阅读平台隐私政策和服务条款。
- 已有账号时，准备用户名或邮箱及密码。

## 推荐阅读顺序

1. 创建账号
2. 选择密码或邮箱验证码登录
3. 登录失败时执行密码找回

## 完成检查

> **用途：** 以下检查是本场景的退出条件，用于判断是否已经取得可观察、可复核的结果，以及是否可以进入下一场景。它不是操作步骤的重复；任一项不满足时，请返回对应功能文档的“常见失败分支”排查。

| 检查项 | 通过标准 |
| --- | --- |
| 1 | 账号通过密码或邮箱验证码成功登录，页面没有持续认证错误。 |
| 2 | 登录后进入正确租户，当前角色、菜单和可见子系统符合预期。 |
| 3 | 找回密码流程可完成，新密码可以登录且旧密码失效。 |
